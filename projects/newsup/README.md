# NewsUp — ニュース英語学習サービス

## プロジェクト概要

**NewsUp** は、英語ニュース記事を素材とした英語学習SaaSです。
毎日配信される本物のニュース記事を読みながら、単語・読解力・リスニング力を同時に強化できます。

### ターゲットユーザー
- 英語をビジネスで使う社会人（TOEIC 650〜800点を目指す層）
- 英語ニュースを読めるようになりたい大学生・社会人
- 現在の英語学習に "飽き" を感じているユーザー

---

## UIプロトタイプの内容

`index.html` にスタンドアロンのインタラクティブUIプロトタイプを作成。

### 実装済みの機能（プロトタイプ）
| 機能 | 説明 |
|------|------|
| 記事リーダー | 難易度タグ・推定読了時間つき |
| ボキャブラリーハイライト | ホバーで日本語訳・例文をツールチップ表示 |
| 理解度クイズ | 4択式・即時フィードバック |
| 単語フラッシュカード | 難易度評価（難しい/まあまあ/簡単） |
| 進捗ウィジェット | 読了記事数・習得語彙数・正答率 |
| 今日のニュース一覧 | カテゴリアイコン・難易度・読了状態表示 |
| 再生速度コントロール | リスニングモード用（UI表示のみ） |

## 技術スタック（想定）

- **フロントエンド**: Next.js + Tailwind CSS
- **バックエンド**: Node.js (Fastify) / Supabase
- **AI**: Claude API（記事要約・クイズ自動生成・難易度判定）
- **音声TTS**: ElevenLabs または Google TTS
- **ニュースソース**: NewsAPI / Reuters RSS
