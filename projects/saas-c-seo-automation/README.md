# SaaS-C: SEO自動化SaaS

## プロジェクト概要

アフィリエイトサイト・コンテンツサイト運営者向けに、native-realで実証済みのSEOパイプラインをSaaSとして提供するB2B/B2Cプロダクト。

Claude APIによる記事・コンテンツ自動生成、内部リンク最適化、SEOレポート自動化をノーコードで提供する。

---

## ターゲットユーザー

| セグメント | 規模 | ニーズ |
|---|---|---|
| アフィリエイトサイト運営者 | 個人〜小規模 | 記事更新工数削減・SEO改善の自動化 |
| Webメディア運営者 | 個人〜SMB | コンテンツ量産とSEO最適化の両立 |
| GitHub Pages静的サイト運営者 | 個人 | native-realと同じパイプラインを自サイトに適用 |
| ブログ×アフィリ複業者 | 個人 | 副業サイトのSEO改善を自動化して本業との両立 |

**主要ペインポイント**:
- 記事を毎週書き続けるのが辛い。ネタ切れ・時間不足
- 内部リンク・メタディスクリプション・見出し構造のSEO最適化を手作業でやっている
- SEOの効果測定（順位・PV推移）を週次でまとめる作業が煩雑

## 収益モデル

### プラン構成（月額サブスク）

| プラン | 月額 | 記事生成上限 | 対象 |
|---|---|---|---|
| Free | ¥0 | 3記事/月 | 試用・小規模 |
| Solo | ¥2,980/月 | 20記事/月 | 個人ブロガー |
| Growth | ¥7,980/月 | 100記事/月 | 本格運用 |
| Agency | ¥29,800/月 | 無制限 + 複数サイト | 代理店・法人 |

### アドオン（従量課金）
- 追加記事: ¥150/記事
- SEOレポートPDF出力: ¥500/回
- 競合分析レポート: ¥1,980/回

### 収益目標
- 3ヶ月目: Solo×30 + Growth×5 = ¥129,300/月
- 6ヶ月目: Solo×80 + Growth×20 + Agency×3 = ¥488,800/月
- 12ヶ月目: MRR ¥1,000,000 達成

### コスト構造
- Claude API: 記事1本あたり約¥30〜80（コスト率 < 5%）
- Vercel: 無料〜¥2,000/月
- Stripe: 売上の2.9%

---

## 競合優位性

1. **native-realで実証済み**: 実際に稼働中のSEOパイプライン（MAU 2,000+）をそのままSaaSに変換
2. **静的サイト対応**: GitHub Pages等の静的サイトにも対応（WordPressプラグインではない）
3. **日本語SEO特化**: 日本語コンテンツ生成・日本語SEO最適化に特化
4. **Claude API活用**: GPT系と差別化、高品質な日本語コンテンツ

---

## コア機能

### 1. 記事自動生成
- トピック候補自動提案（競合分析ベース）
- Claude APIによる記事生成（SEO最適化済み）
- メタディスクリプション・OGP自動生成

### 2. 内部リンク最適化
- 既存記事を分析して内部リンク候補を提案・自動挿入
- アンカーテキスト最適化

### 3. SEOレポート自動生成
- 週次・月次のSEOサマリーレポート（Markdownまたは PDF）
- 記事別パフォーマンス（PV・平均順位・CTR）集計
- Google Search Console API連携

### 4. コンテンツカレンダー
- 記事生成スケジュールの自動管理
- 週次自動実行（native-realのweekly_update.pyと同等）

---

## ロードマップ

- **Phase 1 (MVP)**: 記事生成フォーム + Markdownエクスポート（2週間）
- **Phase 2**: GitHub連携（自動コミット・デプロイ）+ Stripe課金（1ヶ月）
- **Phase 3**: SEOレポート自動生成 + GSC連携（2ヶ月）
- **Phase 4**: 内部リンク最適化 + Agency向け複数サイト管理（3ヶ月以降）
