# affiliate-v2: 英語コーチング専門比較サイト

## 背景

`native-real`（IchiShio/native-real）はアーカイブ済みのため読み取り専用となった。
Tier 1〜2・MAU強化タスクはすべて native-real への書き込みを必要とするため実行不可。

## SEO戦略

### ターゲットキーワード（月間検索数）

| キーワード | 推定月間検索数 | 競合難度 |
|-----------|--------------|---------|
| 英語コーチング おすすめ | 8,000〜12,000 | 高 |
| プログリット 評判 | 3,000〜5,000 | 中 |
| トライズ 口コミ | 2,000〜4,000 | 中 |

| 英語コーチング 効果 | 1,000〜2,000 | 低〜中 |

### コンテンツ差別化

- **診断ウィジェット**: 「あなたにコーチングは向いている？」5問診断 → CTA
- **リアルな比較表**: 料金・期間・サポート頻度・合格率を横並び比較
- **体験談コンテンツ**: 3ヶ月後の変化を時系列で記述（E-E-A-T強化）
- **FAQ**: コーチング特有の疑問（「効果がなかったら？」「転職に役立つ？」）

## 技術仕様

### デザイン方針
- native-real の CSS設計を継承（ヘッダー・カード・ボタンスタイル）
- カラーテーマ: ゴールド系（#f59e0b）でプレミアム感を演出
- モバイルファースト設計

### アフィリエイトリンク管理
- a8.net 経由のリンクを `data-affiliate` 属性で管理
- クリック計測は GA4 イベントトラッキングで対応
- 既存 native-real のアフィリリンク（トライズ: `4AXFOG+786GWQ+3N0O+5YRHE`）を流用可
