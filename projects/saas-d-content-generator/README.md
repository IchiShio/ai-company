# SaaS-D: 英語学習コンテンツ生成SaaS

## プロジェクト概要

英語教室・スクール運営者向けに、Claude APIを活用して英語学習コンテンツ（問題・教材）を自動生成するB2B SaaS。

native-realで実証済みのGrammarUp/ListenUp形式の問題生成ノウハウを、外部の英語教育事業者に提供する。

---

## ターゲットユーザー

| セグメント | 規模 | ニーズ |
|---|---|---|
| 英語教室・英会話スクール | 小〜中規模（生徒10〜200名） | 定期的な教材作成の工数削減 |
| フリーランス英語講師 | 個人 | 自作教材のクオリティ向上・時間節約 |
| 企業内英語研修担当者 | 中小〜大企業 | 社員レベルに合わせたカスタム問題生成 |
| 英語教材制作会社 | SMB | 制作コスト削減・納期短縮 |

**主要ペインポイント**:
- 文法・語彙・リスニング問題の手作りに毎週数時間かかる
- 生徒レベルに合わせた問題の調整が難しい
- 良質な教材を外注すると高コスト

|---|---|---|---|
| Free Trial | ¥0 | 50問/月 | 試用 |
| Starter | ¥3,980/月 | 500問/月 | 個人講師 |
| Professional | ¥9,800/月 | 2,000問/月 | 小規模スクール |
| Business | ¥29,800/月 | 無制限 | 中規模以上 |

### 収益目標
- 3ヶ月目: Starter×10 + Professional×3 = ¥69,200/月
- 6ヶ月目: Starter×30 + Professional×10 + Business×2 = ¥255,000/月
- 12ヶ月目: MRR ¥500,000 達成

### コスト構造
- Claude API: 問題1問あたり約¥1〜3（コスト率 < 10%）
- Vercel: 無料〜¥2,000/月
- Stripe: 売上の2.9%

---

## 競合優位性

1. **native-realでの実績**: 1,000問超の問題生成・公開実績あり
2. **問題品質**: Claude APIによる高品質な英文生成
3. **フォーマット対応**: GrammarUp/ListenUp形式のJSON/CSVエクスポートで既存システムへ即投入
4. **日本語UI**: 国内英語教育市場に特化

---

## ロードマップ

- **Phase 1 (MVP)**: 問題生成フォーム + JSON/CSVエクスポート（2週間）
- **Phase 2**: Stripeサブスク + ユーザー管理（1ヶ月）
- **Phase 3**: 音声合成（ListenUp形式）+ 教材PDF出力（2ヶ月）
- **Phase 4**: API提供 + LMS連携（3ヶ月以降）
