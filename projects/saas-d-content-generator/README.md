# SaaS-D: 英語学習コンテンツ生成SaaS

## プロジェクト概要

英語教室・スクール運営者向けに、Claude APIを使って英語学習コンテンツ（4択問題・穴埋め問題・長文読解）を自動生成し、JSON/CSV形式でエクスポートできるWebサービス。

## ターゲットユーザー

- 英語教室・英会話スクール運営者（個人〜中規模）
- 学習塾の英語担当講師
- オンライン英語コーチ・家庭教師

## 解決する課題

| 課題 | 現状 | 本サービス |
|------|------|-----------|
| 教材作成に時間がかかる | 1問あたり10〜30分 | Claude APIで数秒で生成 |
| 問題の多様性確保が難しい | 同パターンの繰り返し | 文法・語彙・レベル指定で多様生成 |
| LMSへのインポートが手間 | 手入力 | JSON/CSV形式でエクスポート |

## 収益モデル
