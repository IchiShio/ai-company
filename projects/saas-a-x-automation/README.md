# SaaS-A: X（Twitter）自動化SaaS

## プロジェクト概要

X（Twitter）アカウント運用を自動化するB2B/B2C SaaSプロダクト。
Claude APIによるAI投稿生成・自動PDCA・パフォーマンス分析を組み合わせ、
「投稿するだけでフォロワーが増えるX運用」を実現する。

競合多数のジャンルだが、**AI生成+自動PDCA**による差別化で月額サブスクモデルを構築する。

---

## ターゲットユーザー

| セグメント | 規模 | ニーズ |
|---|---|---|
| 副業・フリーランサー | 個人 | 本業をしながらXのフォロワーを伸ばしたい |
| スモールビジネスオーナー | 個人〜SMB | X集客を自動化してリード獲得したい |
| コンテンツクリエイター | 個人 | ネタ切れ・投稿頻度の維持が辛い |
| note・ブログ運営者 | 個人 | note/ブログへのX流入を増やしたい |

**主要ペインポイント**:
- 毎日X投稿を考えるのが辛い。ネタ切れ・文章力への不安
- 投稿してもインプレッションが伸びない。何が効くか分からない
- 複数アカウント管理が煩雑

## 競合優位性（差別化戦略）

競合（Buffer, Hootsuite, Latelyなど）との差別化：

1. **AI生成+自動PDCA**: 単なるスケジューラーではなく、投稿効果を自動分析して次の投稿に反映
2. **日本語X特化**: 日本語ユーザー向けのトレンド把握・日本語最適化プロンプト
3. **Claude APIの高品質日本語**: GPT系より自然な日本語投稿生成
4. **note/ブログ連携**: 既存記事から自動X投稿生成（native-realパイプライン活用）

## コア機能

### 1. AI投稿生成
- テーマ・ジャンル入力でClaude APIが投稿案を複数生成
- 既存記事URL/テキストから自動要約投稿生成
- スレッド形式・単発ツイート・引用リポスト形式に対応

### 2. スケジュール投稿
- カレンダーUI で投稿日時を管理
- エンゲージメント最適時間を自動提案（アカウント分析ベース）
- X API v2経由で自動投稿

### 3. パフォーマンス分析
- インプレッション・エンゲージメント率・フォロワー増減を週次グラフ表示
- 「バズった投稿」の特徴を自動分析してフィードバック
- PDCA: 分析結果を次の投稿生成プロンプトに自動反映

### 4. 複数アカウント管理（Pro以上）
- アカウント切り替えワンクリック
- アカウントごとのペルソナ・投稿スタイル設定

## ロードマップ

- **Phase 1 (MVP)**: AI投稿生成 + X API手動投稿補助（スケジュール機能なし）（2週間）
- **Phase 2**: X APIスケジュール自動投稿 + 基本分析（1ヶ月）
- **Phase 3**: PDCA自動化（分析→次回生成反映）+ 複数アカウント（2ヶ月）
- **Phase 4**: note/ブログ連携、Agency向け機能（3ヶ月以降）

---

## 注意事項・リスク

- **X API制限**: v2 Basic tierはレート制限あり（投稿: 17/15分）。規模拡大時はPro tierへ移行
- **利用規約**: X APIの自動投稿ポリシー遵守必須。スパム判定回避のためコンテンツ品質管理が重要
- **競合**: Buffer・Hootsuite等が強力。価格競争を避け、AI差別化に集中
