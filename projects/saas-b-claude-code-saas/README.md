# SaaS-B: Claude Code活用SaaS（ひとりAI会社モデル販売）

## プロジェクト概要

「ひとりAI会社」モデルをパッケージ化し、Claude Codeを使って事業を自動化したいフリーランス・個人事業主・スタートアップに販売するB2B SaaS。

IchiShioのai-companyで実証済みのAIエージェント組織テンプレート・スキルセット・ワークフローをプロダクト化する。

---

## ターゲットユーザー

| セグメント | 規模 | ニーズ |
|---|---|---|
| フリーランスエンジニア・デザイナー | 個人 | 営業・コンテンツ・SEOを自動化して受注増加 |
| 個人事業主（ライター・コンサル） | 個人 | 反復作業の自動化・副業収入の仕組み化 |
| スタートアップ創業者 | 小規模チーム | 少人数でAIエージェント組織を構築 |
| Claude Code初学者 | 個人 | 実践的なスキルセットを即活用したい |

**主要ペインポイント**:
- Claude Codeの使い方は分かるが、事業に活かすスキル設計・組織化が難しい
- AIエージェントを組み合わせたワークフローをゼロから設計する時間がない
- 自動化の成果（収益・MAU等）を実証したサンプルがない

## 収益モデル

### プラン構成

| プラン | 価格 | 内容 | 対象 |
|---|---|---|---|
| スターターパック | ¥9,800（買い切り） | テンプレートセット（10スキル）+ README | 個人・副業 |
| プロパック | ¥29,800（買い切り） | 全スキルセット + セットアップ動画 + Discordサポート | 本格活用 |
| 月額サポート | ¥4,980/月 | 新スキル毎月追加 + 個別質問対応（非同期） | 継続活用者 |
| セットアップ支援 | ¥49,800（単発） | 1on1でAI会社組織を構築するコンサル | 企業・本気組 |

### 収益目標
- 3ヶ月目: スターター×20 + プロ×5 + 月額×10 = ¥393,800（累計）＋月額¥49,800
- 6ヶ月目: MRR ¥100,000 達成（月額サポート×20 + セットアップ支援月1件）
- 12ヶ月目: MRR ¥300,000（月額×60 + セットアップ支援月2〜3件）

### 販売チャネル
1. **Gumroad**: デジタルダウンロード販売（スターター・プロパック）
2. **note**: 記事内CTAからGumroadへ誘導
3. **X（Twitter）**: oac-writerによる実績ツイート→ランディングページへ
4. **ai-company GitHub Pages**: 実績・ケーススタディページ

---

## 競合優位性

1. **自社実績**: IchiShio ai-companyでの収益自動化事例（数字つき）
2. **即使えるテンプレート**: コピペでセットアップ完了。Claude Code初日から動く
3. **継続アップデート**: スキルを毎月追加。買い切りではなく関係性を構築
4. **日本語特化**: 日本人フリーランス・個人事業主向けの文脈で設計
