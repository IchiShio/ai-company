# プログラミングスクール比較サイト

## プロジェクト概要

native-realモデル（英語学習サービス比較）を複製した、プログラミングスクール比較サイト。
GitHub Pages + 静的HTML で構築し、アフィリエイト収益を狙う。

## ジャンル選定理由

- **高単価**: プログラミングスクールのアフィリ報酬は1件3,000〜30,000円（英会話の10〜30倍）
- **検索ボリューム**: 「プログラミングスクール おすすめ」月間10万+
- **競合状況**: 大手メディアが占拠しているが、特化型・詳細比較で差別化余地あり
- **native-realとのシナジー**: IT系エンジニアが英語学習にも流入する相互送客が可能

## ターゲットユーザー

- 未経験からエンジニア転職を目指す20〜35歳
- 副業・フリーランスでWeb制作を学びたい社会人
- AI・データサイエンス領域への転換を考える会社員

## 収益モデル

| 収益源 | 単価 | 目標CV/月 | 月次収益見込み |
|--------|------|-----------|---------------|
| プログラミングスクール無料相談 | ¥5,000〜¥30,000 | 10件 | ¥50,000〜¥300,000 |
| 関連書籍アフィリ（Amazon） | ¥200〜¥500 | 30件 | ¥6,000〜¥15,000 |

**月次収益目標**: ¥76,000〜¥415,000
