# Factストック

投稿の根拠となるデータ・研究・統計を蓄積する。
投稿作成時は必ずここから引用し、根拠のない数字や主張は使わない。

## ファイル命名規則
- テーマ別: `{テーマ名}.md`（例: `listening.md`, `pronunciation.md`）
- themes/rotation.md の17テーマに対応

## Factのフォーマット
```
### F{番号}: {Factの要約}
- 出典: {論文名/調査名/URL}
- データ: {具体的な数字・結論}
- 投稿での使い方: {どう引用するか}
- 最終確認日: YYYY-MM-DD
```

## ルール
- 出典のないFactは記載禁止
- 「〜と言われている」「一般的に〜」は不可。具体的な出典を明記
- 定期的に出典URLの有効性を確認
