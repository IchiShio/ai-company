# 否定された信念

「効くと思ったが、データで否定された」パターン。同じ過ちを繰り返さないための記録。

### FB1: 発音テーマ×夕方投稿は他テーマと同等のインプレッションが取れる
- 元仮説: なし（データ分析で発見）
- 当初の予測: 発音テーマはどの時間帯でも安定した反応を得られる
- 実際の結果: 発音テーマの夕方投稿（3/19 19:45 1,271imp、3/22夕 1,973imp）はリスニング・学習習慣テーマの平均を大きく下回った。サンプル数は少ない（2投稿）が、発音×夕方の組み合わせは低パフォーマンスの傾向
- 否定日: 2026-03-24
- 教訓: 発音テーマは朝枠に配置し、フックに「数字（効果量、上達率）」を使う。夕方は学習習慣やスピーキングの実践アクション系テーマが向いている可能性がある

### FB2: 共感型フックは常識否定・数字型と同等のインプレッションが取れる
- 元仮説: なし（データ分析で発見）
- 当初の予測: 「覚えが悪い人に共感する」フックは読者の心をつかみ高反応につながる
- 実際の結果: 共感型フック（「それ、あなたのせいじゃないです」）投稿 2,069imp で、数字+意外性型（avg 4,376）や常識否定型（avg 3,220）を大きく下回った
- 否定日: 2026-03-24
- 教訓: フックは「常識否定」か「数字+意外性」を優先。共感型は使う場合でも2行目以降に置き、1行目は常識否定か数字で刺す

### FB3: 「インプット vs アウトプット論争」型フックは低インプレッション
- 元仮説: なし（データ分析で発見）
- 当初の予測: 「研究が答えを出している」という権威訴求フックは知的好奇心を刺激して高反応が得られる
- 実際の結果: 「英語学習は『インプット派 vs アウトプット派』で揉めがちですが、研究はとっくに答えを出しています」（3/27朝）→ 1,251 imp。同週最低値。同日夕の「間隔反復は子どもの話」（2,505 imp）も含め、抽象的な論争提示型は低パフォーマンスの傾向。3/28追加確認: speaking/F3「インプットにアウトプットを組み込む」論争解決型 → 1,251 imp（同週最低値と並ぶ）。サンプル追加で傾向がより確実に
- 否定日: 2026-03-28
- 補強日: 2026-03-30
- 教訓: 「X vs Y 論争に研究が答えを出した」型フックは避ける。論点を提示するより「今すぐやめてください」「〇〇より△△の方が効果量0.92」のように、読者の行動や誤解に直接刺さる具体的フックに置き換える

### FB4: 「半分しかわかってない」型の曖昧な否定フックは直接否定より低imp
- 元仮説: なし（データ分析で発見）
- 当初の予測: 「半分しかわかってない」という間接的な知識提供フックは知的好奇心を刺激して高反応が得られる
- 実際の結果: 3/30朝「シャドーイングはリスニング練習と思ってる人、半分しかわかってない」→ 2,646 imp。同じlisteningテーマの「今すぐやめてください」型（3/22朝 6,161 imp）と比較して57%低い。FB2（共感型フック低迷）に続く曖昧否定型の失敗
- 否定日: 2026-03-31
- 教訓: 常識否定フックは強さと直接性が重要。「半分しかわかってない」「半分は正しい」等の曖昧な表現より「今すぐやめてください」「完全に間違えてます」のような強い直接否定が高imp。P9を参照

### FB5: 条件型フック「〜したいなら〜から」は超低インプレッション
- 元仮説: なし（P10エビデンス収集の過程で発見）
- 当初の予測: 「発音を良くしたいなら口の形より耳から」のような条件提示型は具体的でわかりやすいため高反応が得られる
- 実際の結果: 3/31朝「発音を良くしたいなら口の形より『耳』から。25年分のメタ分析で…」→ 779 imp（週最低値、通常投稿平均の約20%）。同じ発音/F1の強フック版「口の形で発音を直す人、遠回りです」と比較する実験を次週実施予定
- 否定日: 2026-04-01
- 教訓: 「〇〇したいなら△△から」という条件型フックは使用禁止（P10参照）。同じFactでも「〇〇してる人、損してます」「〇〇は今すぐやめてください」のような直接否定型フックに必ず置き換える。Factの価値とフックの強度は独立。良いFactでも弱フックでは埋もれる

### FB6: TOEICスコア系テーマ×P9強フックは一般フォロワーに届かない
- 元仮説: なし（データ分析で発見）
- 当初の予測: TOEICは多くの英語学習者が受験経験を持つため、P9強フックと組み合わせれば幅広い反応が得られる
- 実際の結果: 4/7 morning「TOEIC対策で問題集を何度も読み返してる人、時間を無駄にしてます」(toeic/F-TOEIC-001, P9) → **1,976 imp (C)**。同日夜の対比型テスト (TOEIC 700点台vs800点超え) も **810 imp (D)**。同週 P9×学習習慣(5,753 A)・P9×語彙(10,526 S)と比較して大幅に低い
- 否定日: 2026-04-09
- 教訓: TOEICは「英語学習者の中でもTOEICを受験中の層」にしか刺さらない。@ichi_eigoのフォロワー層（英語学習全般に関心）には母数が狭い。TOEIC特化コンテンツは最大でも月1〜2本程度に抑制し、メインは「リスニング・語彙・学習習慣・スピーキング」の普遍テーマを使う

### FB7: 対比型フック×抽象的な意識差（精神論・マインドセット）は機能しない
- 元仮説: H8（対比型フック仮説）
- 当初の予測: 「AとBの違い」型の対比フックはP9型と同等以上のインプレッションを出せる
- 実際の結果: 4/7 night「TOEIC 700点台の人と800点超えた人、勉強量より"目標の意味"を知ってるかどうかの差」→ **810 imp (D)**。同じH8テスト①の対比型「2,000語を知っている人と7,000語族を持っている人」(6,505 A) と対照的。抽象的な意識差（目標の意味）vs 具体的な知識・スキル格差（語彙数）で大きく分かれた
- 否定日: 2026-04-09
- 教訓: 対比型フックは「測定可能な具体格差（語彙数・効果量・スコア点数・学習時間）」を使う。「意識の差」「取り組み方の差」「考え方の差」など定性的な格差は機能しない（FB3の論争型と同様）
