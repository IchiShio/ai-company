# 投稿テンプレート

データで進化するテンプレート集。各テンプレートに使用回数と平均エンゲージメント率を追跡する。

---

## @ichi_eigo 用テンプレート

### T1: 常識否定 → 解決策提示（メインテンプレート）

```
[常識否定のフック（1行目）]

[なぜそれが間違いか（2-3行）]

[正しいアプローチ（2-3行）]
```

- 文字数: 130〜140字
- ハッシュタグ: なし
- 絵文字: なし
- 使用回数: 0 / 平均エンゲ率: -

### T2: 具体ノウハウ型

```
[数字入りフック]

[ステップ1]
[ステップ2]
[ステップ3]
```

- 使用回数: 0 / 平均エンゲ率: -

### T3: 体験ストーリー型

```
[体験告白フック]

[ビフォー状態]
↓
[ターニングポイント]
↓
[アフター＋学び]
```

- 使用回数: 0 / 平均エンゲ率: -

## 更新ルール

- 週次レビュー時に使用回数と平均エンゲージメント率を更新
- 平均エンゲ率がBランク以下のテンプレートは改善案を検討
- 新しいパターンでSランクが出たらテンプレートとして追加
