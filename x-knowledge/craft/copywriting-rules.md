# コピーライティングルール

## @ichi_eigo

- 文字数: 130〜140字（ハッシュタグなし、絵文字なし）
- 構造: 「常識否定 → 解決策提示」が基本形
- 4ステップ: Fact選定 → Draft → ファクトチェック → Final
- 投稿枠: 毎日 7:30 と 18:30（JST）
- トーン: 権威的すぎず、友人のような親しみやすさ
- 改行: 2〜3行ごとに空行を挟む

## @careermigaki

- 形式: IODフレームワーク（Phase 1: 情報提供重視）
- マルチパートスレッド形式
- テーマ: STAR面接、レジュメ、報酬体系、求人市場データ
- 数値データは必ずファクトチェック
- トーン: プロフェッショナルだが温かみのある語り口

## 共通ルール

- 1行目（フック）で勝負が決まる。ここに最大の注意を払う
- 抽象的な表現より具体的な数字・事例
- 「〜しましょう」より「〜してください」（直接的な語りかけ）
- 読者の悩みに寄り添ってから解決策を提示する
