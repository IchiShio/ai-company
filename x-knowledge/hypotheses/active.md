- 成功指標: 数字ありの平均インプレ ÷ 数字なしの平均インプレ ≥ 1.3
- テスト方法: 同テーマで数字あり/なしを交互に投稿して比較
- 開始日: 2026-03-18
- 投稿数: 32/10（大幅超過・継続監視中）4/2追加: 英語字幕×「実は〇〇型」12,487 imp (S)、シャドーイング×「損してます」3,741 imp (B)。「実は〇〇型」が数字+意外性の新高値（H4として分離登録）
- 現在のデータ: 比率 推定1.8+（弱フック除外）。4/2「実は〇〇型」12,487 imp がリスニングテーマ最高値更新。シャドーイング×中フック「損してます」3,741 impはP10の中フック帯に一致
- ステータス: 確認済み → P1参照（P7・P8・P9・P10も追加、FB5新規追加、H4新規追加）

### H2: 朝7:30は夕方18:30よりエンゲージメント率が高い

- 成功指標: 7:30の平均エンゲ率 ÷ 18:30の平均エンゲ率 ≥ 1.2
- テスト方法: 同品質の投稿を両枠に均等配分し比較
- 開始日: 2026-03-18
- 投稿数: 36/20（大幅超過・継続監視中）4/2追加: 英語字幕×リスニング(枠不明) 12,487 imp (S)、シャドーイング(枠不明) 3,741 imp (B)。枠情報なし（新フォーマットにtime_slot未収録）
- 現在のデータ: 強フック朝avg ~5,000+ / 弱フック朝avg ~900。フック強度制御後の時間帯効果比率 推定1.5+。P4（スピーキング夕方可）・FB1（発音朝固定）・FB5（条件フック禁止）を維持
- ステータス: 確認済み → P2参照（注: フック強度が時間帯より優先度高いことを確認。弱フックは朝でも低い）

### H3: 常識否定型フックは断言型フックよりリプライ率が高い

### H4: 「実は〇〇型」意外性+行動否定フックは「今すぐやめて」型よりRT率が高い
- 仮説: 「実は、[現在やっている行動]しても、効果は[定量的損失]です」型フックは、P9（今すぐやめてください）型より平均RT率が1.3倍以上高い
- 成功指標: 「実は〇〇型」平均RT率 ÷ P9型平均RT率 ≥ 1.3（かつ imp も 5,000+）
- テスト方法: 「実は、[行動]しても、[定量損失]です」型フックを3〜5件テストし、P9型同テーマと比較
- 開始日: 2026-04-02
- 投稿数: 2/5（投稿済み2件 + 予約済み2件: 4/13朝・4/18夕）
- 現在のデータ:
  - テスト①「実は、英語字幕を付けてリスニングしても…」→ **31,239 imp (S)** / RT=21 / BM=174 / RT率0.067%（4/5確定値）
  - テスト②「実は、独り言で英語を練習しても…」→ **7,679 imp (A)** / RT=3 / BM=26 / RT率0.039%（4/5確定値）
  - 比較対象P9型（今すぐやめて×リスニング）: 6,161 imp / RT=4 / RT率0.065%
  - テスト①の RT率比: 0.067÷0.065=1.03（成功指標1.3届かず）
  - テスト②の RT率比: 0.039÷0.065=0.60（RT率はP9型を大幅下回る）
  - テーマ依存分析: ①リスニング字幕(普及行動×高否定感) → S級。②スピーキング独り言(否定感薄め) → A級。テーマ選択がフック効果の鍵と推定
  - 次回予定: 4/13朝「実は、ゆっくり英語を聴き続けても…」listening/F3（リスニング系再テスト）
  - 4回目: 4/18夕「実は、同じ1人のネイティブだけで発音練習しても…」pronunciation/F2
- **4/6週次PDCA更新**: テスト①確定値 31,239 imp (S) / RT率0.067%。テスト②確定値 7,679 imp (A) / RT率0.039%。impはともに高値だがRT率はP9比で下回る。「実は〇〇型」はRT率よりinpression獲得型フックの可能性が高まった。4/13・4/18の残り3テスト結果で最終判定。
- ステータス: テスト中（確定値2/5）

### H5: 同テーマを7日以内に再投稿すると初回比30%以上インプレッション低下する
- 仮説: 同一ファクトまたは隣接サブテーマ（例: 分散学習・間隔反復・記憶定着は同一クラスタ）を7日以内に再投稿すると、初回投稿比で30%以上のインプレッション低下が発生する
- 成功指標: 再投稿のinp ÷ 初回投稿のinp ≤ 0.7 が3ケース以上で確認
- テスト方法: 同テーマ再投稿事例を記録し比較
- 開始日: 2026-04-06
- 投稿数: 1/3（事例1件：4/5「間隔復習×大人」= 1,491 imp、4/3初回比22%）
- 現在のデータ:

- ステータス: テスト中（1/3）

### H7: 「実は〇〇型」の効果はテーマの行動普及度に比例する
- 仮説: 「実は、[行動]しても、[損失]です」型フックの効果は、フォロワーの中でその行動を実際に行っている人の割合（行動普及度）に比例する。普及度S（字幕学習・スロー再生・単語帳暗記 = ほぼ全員が経験）は10,000+ imp、普及度B（独り言練習・4/3/2 = 一部のみ）は5,000〜8,000 impにとどまる
- 成功指標: 行動普及度S（字幕・スロー再生・単語帳暗記）での「実は〇〇型」avg imp ≥ 15,000 が3件以上確認される
