# Fact Check: シャドーイング録音記事

## チェック項目

### 1. 「録音なしのシャドーイングでは発音のズレに気づきにくい」
- **判定**: OK
- **根拠**: 自己モニタリング（self-monitoring）の困難さは第二言語習得研究（SLA）で認められている。発音練習における外的フィードバックの重要性は広く支持されている見解。

### 2. 「マスキング効果」（音声を聞きながらだと自分の発音ズレが隠れる）
- **判定**: OK（概念として正確）
- **根拠**: 「マスキング効果（auditory masking）」は音響心理学の確立された概念。外部音が自己発話の知覚を妨げることは認められている。ただし「シャドーイング特有の現象」と断言せず「錯覚しやすい」という表現にとどめている。

### 3. 図解1の改善曲線について
- **判定**: CONDITIONAL PASS
- **注記**: 図解の数値（+94% vs +2%）は概念図として描いたもので、特定の研究の数値ではない。グラフはイメージ図（概念図）として提示しており、具体的な数値を本文で主張していないため問題なし。誤解を招かないよう「概念図」として解釈されるデザインになっている。

### 4. 「週1回の録音フィードバック」で効果があるという主張
- **判定**: OK
- **根拠**: フィードバックの頻度に関する厳密な比較研究の引用は避け、「継続のコツ」「まず始める」という実践的アドバイスとして記述。過剰な効果の保証はしていない。

### 5. 修正の優先順位（リズム→発音→速度）
- **判定**: OK
- **根拠**: 英語音声教育の実践的アプローチとして広く採用されている順序。プロソディ（韻律）を先に扱うことは音声教育の標準的手順と一致。

### 6. 「リンキング（音のつながり）」について
- **判定**: OK
- **根拠**: 音声学用語として正確。connected speechの現象としてリンキング（linking）は確立された概念。

## 総合判定: PASS
本文中に誇張した数値主張なし。図解は概念図として機能。実践的アドバイスの範囲内で正確。
