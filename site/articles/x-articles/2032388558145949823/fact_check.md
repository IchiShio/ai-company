# Fact Check: チャンク読み記事

## チェック項目

### 1. 「ネイティブはチャンク（意味かたまり）単位で読む」
- **判定**: OK
- **根拠**: 心理言語学の読解研究（Just & Carpenter, 1987 など）において、習熟した読者は語彙的フレーズや句単位で処理することが示されている。チャンク処理は語用論・読解研究の標準的知見。

### 2. 「視点の停止回数（フィクセーション）が熟練者は少ない」
- **判定**: OK
- **根拠**: 眼球運動研究（Eye-tracking studies）は読解研究の主要手法。熟練読者のフィクセーション頻度が低く、スパン（1回の停止で処理できる文字数）が広いことは広く認められている事実。

### 3. 「ワーキングメモリの負荷が読速に影響する」
- **判定**: OK
- **根拠**: 認知心理学の作業記憶モデル（Baddeley, 2000）で確立されている。読解中の認知負荷が高いと処理速度が低下することは実証済み。

### 4. 「スラッシュ・リーディングはチャンク読み練習に有効」
- **判定**: OK（一般的な英語教育実践として広く採用されている手法）
- **根拠**: 日本の英語教育でも採用されている実践的手法。過度な効果の主張はなし。
