# Fact Check: 英語リズム記事

## チェック項目

### 1. 「英語は強勢拍リズム（stress-timed rhythm）言語」
- **判定**: OK
- **根拠**: 音声学の確立された分類（Pike, 1945; Abercrombie, 1967）。英語はstress-timed、日本語はmora-timed言語であることは音声学の標準的知識。

### 2. 「個別音より文全体の強弱リズムが伝達性に影響する」
- **判定**: OK（概ね正確）
- **根拠**: 音声コミュニケーション研究でリズム・プロソディが理解度に大きく寄与することは認められている（Tajima et al., 1997 など）。ただし「R/Lより重要」という絶対的な主張は避け、「も重要」「最も頼っている」という表現で記述。
- **注記**: 元の投稿も「RとLの違いより」という相対表現を使っており、適切。

### 3. 「日本語はモーラ拍リズム」
- **判定**: OK
- **根拠**: 日本語音声学の標準的分類。

### 4. 「内容語=強く、機能語=弱く」のルール
- **判定**: OK
- **根拠**: 英語音声学・音調論の基本規則。多数の教科書（Murphy "Pronunciation Pairs" 等）で確認できる。

### 5. 機能語の弱形（短縮形）
- **判定**: OK
- **注記**: "wanna"（want to）、"gonna"（going to）は口語での短縮形。インフォーマルな文脈でのみ使われることを示唆する記述にとどめている。

### 6. 図解内の英文・発音表記
- "I want to learn English" の強弱パターン ✓
- 冠詞の弱形: a → /ə/、an → /ən/、the → /ðə/ ✓
- 分類（名詞/動詞/形容詞/副詞 = 内容語）✓
- 分類（冠詞/前置詞/接続詞/代名詞/助動詞 = 機能語）✓

## 総合判定: PASS
音声学の確立された概念に基づく正確な内容。誇張なし。
