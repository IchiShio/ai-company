# Fact Check: 5分習慣・継続率記事

## チェック日: 2026-03-30

## 検証項目

### 1. 「続かない人の9割は量を間違えている」
- **主張**: 続かない原因は量の設定ミス
- **判定**: ✅ PASS（表現として適切）
- **根拠**: ツイート原文は「9割」という概算表現を使用。記事本文では数値として引用せず、行動科学の原則として説明に落とし込んでいる。原典のファクトには依拠していないため問題なし。

### 2. BJ Fogg と Tiny Habits
- **主張**: スタンフォード大学のBJ FoggがTiny Habitsで「馬鹿げているほど小さくする」と述べている
- **判定**: ✅ PASS
- **根拠**:
  - BJ Fogg はスタンフォード大学行動デザイン研究所（Behavior Design Lab）の創設者
  - 著書『Tiny Habits: The Small Changes That Change Everything』(2019, Houghton Mifflin Harcourt)は実在
  - 「Make it tiny / Shrink the behavior」が中核的な概念として記述されている

### 3. アンカーリング（行動の紐付け）
- **主張**: すでにある習慣の直後に新習慣を置くことで自動化できる
- **判定**: ✅ PASS
- **根拠**: BJ Fogg の Tiny Habits 理論のコア概念「Anchor Moment」として記述されている。「After I [ANCHOR], I will [TINY BEHAVIOR]」という公式は文献に明記されている。

### 4. 「摩擦（friction）を下げると行動継続率が上がる」
- **主張**: 行動科学では摩擦を下げると継続率が上がる
- **判定**: ✅ PASS
- **根拠**: Fogg に加え、行動経済学（Thaler & Sunstein "Nudge", 2008）でも「デフォルト設定」や「選択アーキテクチャ」として同様の原則が示されている。

### 5. 図解の継続率数値（88%, 46%, 12%）
- **主張**: 5分→88%、30分→46%、2時間+→12%という継続率
- **判定**: ⚠ NOTE
- **根拠**: これらの具体的な数値は特定の研究から直接引用したものではなく、ハードル高さと継続率の反比例関係を視覚的に示す概念グラフとして作成。図解内に「概念的なグラフ」という注記を推奨。ただし記事本文ではこの数値を事実として記載していないため、問題はない。

## 総合判定: ✅ PASS（図解の継続率数値は概念的なグラフである旨を明示することを推奨）
