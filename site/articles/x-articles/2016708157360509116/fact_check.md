# Fact Check Report
## 投稿ID: 2016708157360509116

### 検証項目

#### ✅ 「英語が話せない人の9割が翻訳経由で考えている」
- **判定**: 表現として許容範囲（ただし出典なし）
- **根拠**: この正確な数値を示す学術研究は確認できない。元投稿（Xの文体）では比喩的な数字として使用されており、記事本文では「9割」という表現を引用していない。記事は「多くの学習者」という表現に修正済みで問題なし。

#### ✅ 「ネイティブは日本語を経由しない」
- **判定**: 正確
- **根拠**: 第一言語話者の言語処理には媒介言語が存在しないことは言語心理学の基本的知見（Levelt 1989 Speaking; Bialystok 2001）。

#### ✅ 「翻訳がうまくなってもゴールではない」
- **判定**: 正確
- **根拠**: KrashenのAcquisition-Learning仮説（1982）に基づけば、意識的な翻訳学習（Learning）は自然な産出（Acquisition）に直接つながらない。コミュニケーション能力の目標は自動的処理（automaticity）にあることはSLAの共通見解（DeKeyser 2007）。

#### ✅ 「Step4段階のロードマップ」
- **判定**: 筆者独自の整理であり学術的根拠を要しない
- **根拠**: 段階的習熟モデルとしてはDreyfus model（1980）やFluency建設モデルと整合的。4段階という区切りは説明便宜上の設定であり、誇張なし。

#### ✅ 「BBC Learning English / VOA / TED-Ed」の教材推奨
- **判定**: 正確・適切
- **根拠**: いずれも無料・公開の英語教育リソース。BBCとVOAは英語母語話者向け媒体、TED-Edは教育動画として世界的に認知されている正規コンテンツ。

### 総合判定
**✅ PASS** — 主要主張は言語習得研究と整合的。独自の数値・断定的表現なし。
