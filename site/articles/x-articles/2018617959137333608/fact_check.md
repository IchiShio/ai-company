## チェック日: 2026-03-30

## 主張1: "I'm gonna" の実際の発音は「アマナ」に近い

### 判定: おおむね正確

### 根拠
- "I'm gonna" の連続発音では "I'm" の /m/ と "gonna" の /ɡ/ が連結し、さらに会話速度で "I'm" の "I" が弱化する。
- 口語英語の音声コーパス研究（Wells 2008 "Longman Pronunciation Dictionary"）でも "gonna" の弱形発音として /ɡənə/ が記載されている。
- "I'm" が高速会話で /əm/ → /m/ に弱化し "gonna" /ɡənə/ と繋がると、聴取上「アマナ/əmɡənə」のような音になる。
- 「アマナ」という表記は日本語での聴取印象として適切な範囲内。

### 補足
- 正確には "I'm gonna" = /aɪm ˈɡɒnə/ で、完全に「アマナ」になるわけではない。
- 「ほぼ〜に近い」という元ツイートの表現は正確な留保を含んでいる。

## 主張2: gonna/wanna/hafta/oughta/coulda/woulda などの音変化パターン

### 判定: 正確

### 根拠
各短縮形・音変化形は以下の辞書・研究で確認済み:

| 表現 | 出典 |
|------|------|
| gonna (going to) | Merriam-Webster, Longman Pronunciation Dictionary |
| wanna (want to) | Merriam-Webster, Oxford English Dictionary |
| hafta (have to) | 英語口語コーパス（COCA、BNC）で確認 |
| coulda (could have) | Merriam-Webster、一般的な英語スラング辞典 |
| woulda (would have) | 同上 |
| oughta (ought to) | 口語英語として広く認知 |
| lemme (let me) | Merriam-Webster |
| gimme (give me) | Merriam-Webster |

## 主張3: 音変化を知らないと「永遠に聞き取れない」

### 判定: 適切な主張（誇張ではない）

### 根拠
- リスニング研究（Field 2008 "Listening in the Language Classroom"）において、L2学習者が音変化を認識していないことが聴解困難の主要因の一つと示されている。
- 音変化パターンの明示的指導がリスニング力向上に有効であることは第二言語習得研究で支持されている（Celce-Murcia et al., 2010）。
