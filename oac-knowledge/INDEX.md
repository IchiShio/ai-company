# oac-knowledge INDEX

one-ai-company部門（@one_ai_company）の知識ベース。

## ディレクトリ構成

```
oac-knowledge/
  INDEX.md              <- このファイル
  craft/
    voice.md            <- ブランドペルソナ・トーン定義
  facts/
    ai-company-market.md <- 市場データ・競合情報
  hypotheses/
    active.md           <- テスト中の仮説
  learnings/
    patterns.md         <- 確認済み勝ちパターン
    false-beliefs.md    <- 否定された信念
  posts/
    oac-log.csv         <- X投稿パフォーマンスログ
```

## タスク別の読み込みパス

### X投稿を作成する

1. `craft/voice.md` -- トーン・ペルソナ
2. `learnings/patterns.md` -- 勝ちパターン（適用）
3. `learnings/false-beliefs.md` -- 否定された信念（回避）
4. `hypotheses/active.md` -- テスト中の仮説（意図的に試す）

### 分析する

1. `posts/oac-log.csv` -- 投稿パフォーマンスデータ
2. `facts/ai-company-market.md` -- 市場データ・競合情報
3. `hypotheses/active.md` -- 仮説検証用
4. `learnings/patterns.md` -- 既知パターンとの比較

### 戦略を検討する

1. `facts/ai-company-market.md` -- 市場状況
2. `learnings/patterns.md` -- 何が効いているか
3. `learnings/false-beliefs.md` -- 何が効かないか
4. `hypotheses/active.md` -- 次に試すこと
