## 実装手順（Beehiivでの設定方法メモ）

### 1. Automation（自動化）の設定

1. Beehiivダッシュボード → **Automations** → **New Automation**
2. トリガー: **New Subscriber** を選択
3. 各Dayのメールをステップとして追加:
   - Step 1（Day 0）: **Send Email** → 遅延: 即時（0分）
   - Step 2（Day 1）: **Wait** → 1日後 → **Send Email**
   - Step 3〜8（Day 2〜7）: 同様に1日ずつ遅延設定

### 2. 各メールの作成

1. **Posts** → **New Post** → メール形式で作成
2. タグ付け: `step-mail` `7day-training`
3. 配信設定: **Automation専用**（通常配信から除外）

### 3. フォームの埋め込み設定

埋め込み用iframeコード（newsletter/index.htmlに設置済み）:
```html
<iframe
  src="https://embeds.beehiiv.com/pub_9b2ae59d-7a25-49f8-be58-793c82e4a498"
  data-test-id="beehiiv-embed"
  width="100%"
  height="320"
  frameborder="0"
  scrolling="no"
></iframe>
```

### 4. Welcome Emailの有効化

- **Settings** → **Welcome Email** → 有効化
- ※ Automationとの二重送信に注意。どちらか一方に統一する。
- 推奨: Welcome EmailをオフにしてAutomationのDay 0で送信。

### 5. Analyticsの確認

- **Automations** → 各ステップの開封率・CTR確認
- 週次でKPI（開封率30%以上、CTR5%以上）を確認
- CTR低いステップは件名・本文・CTAを改善してA/Bテスト

### 6. A8.netトラッキングピクセルの設置

Beehiivのリッチテキストエディタ → **HTML block** を挿入 → 以下を貼り付け:
```html
<img border="0" width="1" height="1" src="https://www1X.a8.net/0.gif?a8mat={コード}" alt="">
```
※ `{コード}` は各サービスのA8.netパートナー管理画面から取得。
