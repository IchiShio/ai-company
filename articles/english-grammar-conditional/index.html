英語の条件文（if節）の使い方【仮定法現在・過去・過去完了の違いと例文】

目次

  1. ゼロ条件文（Zero Conditional）— 普遍的な事実
  2. 第1条件文（First Conditional）— 現実的な条件
  3. 第2条件文（Second Conditional）— 仮定法過去
  4. 第3条件文（Third Conditional）— 仮定法過去完了
  5. 混合条件文と unless の使い方
  6. 4つの条件文まとめ比較

「If I were you, I would study harder.」と「If I study hard, I will pass.」の違いがわかりますか。どちらも if を使っていますが、表す意味とそれぞれの動詞の形が大きく異なります。英語の条件文は4つのパターンに分けて理解すると、使い分けが一気に楽になります。本記事では4パターンを例文と図解で整理します。

ゼロ条件文（Zero Conditional）— 普遍的な事実

ゼロ条件文は、科学的事実や普遍的な真理、習慣的な出来事を表すときに使います。「〜すると、必ず〜になる」という関係で、条件と結果が常に成立します。

構造

If + 現在形, 現在形（または命令形）

If you heat water to 100°C, it boils.

水を100度に加熱すると、沸騰します。（科学的事実）

If I don't sleep enough, I feel tired the next day.

十分に眠らないと、翌日疲れを感じます。（習慣的な出来事）

If you press this button, the door opens.

このボタンを押すと、ドアが開きます。（操作手順）

ゼロ条件文では if の代わりに when を使うことも多く、意味はほぼ同じです。科学や自然の仕組みを説明する文脈でよく使われます。

第1条件文（First Conditional）— 現実的な条件

第1条件文は、現実に起こりうる未来の条件とその結果を表します。「もし〜するなら（する可能性がある）、〜だろう」という現実的な想定です。

構造

If + 現在形, will + 動詞の原形

If it rains tomorrow, we will cancel the picnic.

明日雨が降ったら、ピクニックをキャンセルします。（現実的な可能性）

If you study hard, you will pass the exam.

一生懸命勉強すれば、試験に合格するでしょう。

If she calls me, I will tell her the news.

彼女が電話してきたら、そのニュースを伝えます。

注意点として、if 節の中は will を使わず現在形にするのが基本ルールです。「If it will rain…」とは言いません。ただし丁寧に依頼する場合（"If you will please wait…"）は例外的に will が入ることがあります。

will の代わりに can・may・might・should などの助動詞も使えます。

If you finish early, you can leave the office.

早く終わったら、オフィスを出てもいいですよ。（許可）

第2条件文（Second Conditional）— 仮定法過去

第2条件文は、現在の事実に反する仮定、または起こりにくい未来のシナリオを表します。日本語の「もし〜だったとしたら」に相当します。

構造

If + 過去形, would + 動詞の原形

If I were rich, I would travel the world.

もしお金持ちだったら、世界中を旅するのに。（現実は違う）

If she lived closer, we would meet more often.

彼女がもっと近くに住んでいたら、もっと会えるのに。

What would you do if you won the lottery?

もし宝くじが当たったら何をしますか。

be 動詞の扱いについて重要なポイントがあります。仮定法過去では、主語が I / he / she / it であっても were を使うのが正式とされています（"If I were you…"）。ただし口語では "If I was you…" という形も広く使われています。

would の代わりに could・might も使えます。

If I had more time, I could learn a new language.

もっと時間があれば、新しい言語を学べるのに。（能力の可能性）

第3条件文（Third Conditional）— 仮定法過去完了

第3条件文は、過去の事実に反する仮定を表します。「あのとき〜していたら、〜だったのに」という後悔や反省、または単純に過去の別の展開を想像するときに使います。

構造

If + 過去完了形（had + 過去分詞）, would have + 過去分詞

If I had studied harder, I would have passed the exam.

もっと勉強していたら、試験に合格していたのに。（過去の後悔）

If she had taken the earlier train, she would not have been late.

もっと早い電車に乗っていたら、遅刻しなかっただろうに。

If they had invested earlier, they would have made a lot of money.

もっと早く投資していたら、たくさん儲けていただろうに。

could have / might have も would have の代わりに使えます。

If we had left earlier, we might have caught the last train.

もっと早く出発していたら、終電に乗れたかもしれない。

混合条件文と unless の使い方

混合条件文（Mixed Conditional）

第2条件文と第3条件文を組み合わせた形も使われます。過去の仮定が現在の状況に影響するケースです。

If I had studied medicine, I would be a doctor now.

医学を勉強していたら、今頃医者になっていただろうに。（過去の仮定 → 現在の結果）

If she were more experienced, she would have gotten the job.

彼女がもっと経験豊富なら、あの仕事を獲得できていただろうに。（現在の仮定 → 過去の結果）

unless の使い方

unless は "if … not" とほぼ同じ意味で、「〜しない限り」と訳せます。第1条件文でよく使われます。

Unless you hurry, you will miss the bus.

急がない限り、バスに乗り遅れますよ。（= If you don't hurry, you will miss the bus.）

I won't go unless you come with me.

あなたが一緒に来ない限り、私は行きません。

4つの条件文まとめ比較

4つの条件文を整理すると、以下の表のようになります。

種類 if 節 主節 ニュアンス
ゼロ条件文 現在形 現在形 普遍的事実・習慣
第1条件文 現在形 will + 原形 現実的な未来の条件
第2条件文 過去形 would + 原形 現在の事実に反する仮定
第3条件文 過去完了形 would have + 過去分詞 過去の事実に反する仮定

「If I win the lottery, I will buy a house.」と「If I won the lottery, I would buy a house.」の違いは微妙に見えますが、前者は「もしかしたら当たるかもしれない」という現実的期待を含み、後者は「まず当たらないだろうが、もしも…」という非現実的な仮定を示します。この使い分けが自然にできるようになると、英語表現の幅が大きく広がります。
