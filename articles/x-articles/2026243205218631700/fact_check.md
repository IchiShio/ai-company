- **クレーム**: 筋トレと同様に、英語学習も「少しずつ難しい素材」に触れることで成長する
- **判定**: ✅ 事実
- **根拠**: Progressive Overload はスポーツ科学・筋生理学で確立された原則（ACSM ガイドライン等）。応用行動分析・教育心理学でも学習の漸進性は広く支持されている。英語教育への適用は研究上も妥当。

### 2. ヴィゴツキーの最近接発達領域（ZPD）
- **クレーム**: 学習が最も効率よく起きるのは「今の能力より少し難しい帯域」
- **判定**: ✅ 事実
- **根拠**: ヴィゴツキー（1978）"Mind in Society" が原典。教育心理学の標準的概念として広く引用される。英語教育研究でも適用事例多数。

### 3. 「7〜8割理解できる素材が最適」
- **クレーム**: 理解度70〜80%の素材が最も効率よく伸びる
- **判定**: ✅ ほぼ事実（概念的に妥当）
- **根拠**: Krashen（1985）の「i+1仮説」（Comprehensible Input Hypothesis）と整合する。ただし「正確に7〜8割」という数値自体に実験的根拠があるわけではなく、概念的目安として提示するのが適切。記事はそのような書き方をしている。

### 4. 3ゾーンモデル（快適・学習・パニック）
- **クレーム**: 図解に示した3ゾーンの概念
- **判定**: ✅ 事実（モデルとして確立）
- **根拠**: Vygotsky ZPD + パニックゾーン概念は教育界で広く使われる概念モデル。ただし「パニックゾーン」という命名はポピュラー心理学的用法で、厳密な学術論文名称とは異なる場合があるため、図解に「概念モデル」の注記を付けることが望ましい（記事ではモデルとして提示している）。
