## 検証項目

### 1. 内容語・機能語の分類
- **主張**: 英語の語には「内容語（動詞・名詞・形容詞・副詞）」と「機能語（前置詞・冠詞・代名詞・助動詞）」がある
- **判定**: ✅ PASS
- **根拠**: 言語学の確立された概念（content words vs. function words）。Quirk et al. "A Comprehensive Grammar of the English Language"等で標準的に使用される分類。

### 2. ネイティブは機能語を弱く発音する
- **主張**: ネイティブが話すとき、機能語はほぼ聞こえないほど弱く発音される
- **判定**: ✅ PASS
- **根拠**: 英語の強勢タイミング（stress-timed rhythm）の特性。内容語が強勢を受け、機能語は弱形（weak form）で発音されることは音声学の基本原則（Cruttenden 2014 "Gimson's Pronunciation of English"等）。

### 3. 「内容語だけで文意の70%以上が把握できる」
- **主張**: 動詞＋名詞の2点に絞ると文意の70%以上が見える
- **判定**: ⚠️ SOFT PASS
- **注意**: 「70%以上」という具体的な数値の出典は特定できない。ただし、内容語が意味の中核を担うという言語学的事実に基づく合理的な推定であり、誇張的な表現ではない。記事内では「見えてくる」という表現で断定を避けている。

### 4. 音の連結・脱落（going to → gonna など）
- **主張**: ネイティブは "going to" を "gonna" と発音する
- **判定**: ✅ PASS
- **根拠**: 英語の音変化（connected speech）の典型例。gonna / wanna / gimme 等は広く認められた口語的短縮形。

### 5. 動詞の例示（came / told / got / made）
- **主張**: これらが「強く発音される動詞」の例として挙げられている
- **判定**: ✅ PASS
- **根拠**: すべて英語の一般的な過去形動詞。内容語として強勢を受ける通常の動詞。

### 6. 「1ヶ月続けると確実にリスニングが楽になる」
- **主張**: 1ヶ月で効果が感じられる
- **判定**: ⚠️ SOFT PASS
- **注意**: 「確実に」は個人差があるため断言は誇張になりうる。ただし記事内では「感覚が出てくる」という柔らかい表現で述べており、保証ではなく体験として書かれている。許容範囲内。
