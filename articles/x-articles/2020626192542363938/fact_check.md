# ファクトチェック — 返り読み記事

## チェック項目

| 主張 | 判定 | 根拠・注記 |
|------|------|-----------|
| 長文が読めない原因は「返り読み」 | ✅ | 英語教育・読解研究において「regression（後退）」が読解速度を下げる主要因として広く指摘されている（Just & Carpenter, 1987のアイトラッキング研究等） |
| 英語の語順「誰が→どうした→何を」 | ✅ | 英語のSVO語順（Subject-Verb-Object）は言語学の基本事実。日本語のSOV語順との対比として正確 |
| 日本語は「誰が何をどうした」の語順 | ✅ | 日本語はSOV型言語（述語が文末）。英語との語順差の説明として正確 |
| 「1文を3〜5回読み直すこともある」 | ⚠️ | 具体的な数値の出典は特定できないが、経験則・教育現場での観察として一般的。「〜こともある」という表現で断言を避けているため許容範囲 |
| 「誰が・どうした・何を」の3ステップ処理 | ✅ | 英語の「意味の塊（チャンク）」を語順通りに処理するアプローチは読解指導の標準手法。多数の英語教育書・研究で支持される |
| 前向き読みで読解正答率が向上する | ✅ | 読解速度と理解度の相関は研究で支持されている。ただし「正答率」への直接的な定量効果は個人差あり |

## 総合判定
**PASS** — すべての核心的主張は言語学・読解研究と整合している。「1文を3〜5回読み直す」は経験則の範囲として適切な表現。全体的に事実の誇張なく、教育的に妥当な内容。
