# Fact Check Report
tweet_id: 2030214374082892118

## 検証項目

### 1. 「日本人は中高で約3,000語を学んでいる」
**判定: 正確（公的基準に準拠）**
- 文部科学省 学習指導要領（2021年改訂）: 中学校1,600〜1,800語 + 高校で追加学習 = 合計約3,000〜4,000語が目標語彙数として設定されている。
- 「約3,000語」は保守的かつ正確な表現。

### 2. 「アクティブ語彙は約300語程度」
**判定: 概念的に正確（実証値に個人差大）**
- 日本人英語学習者の産出語彙数に関する研究は多数あり、平均的な大学生でも意識的にアウトプットできる語彙は認識語彙の10〜15%程度という報告がある。
- 「300語」は推計値であり、実際は個人差が大きい。記事内では「約300語程度と言われる」と留保表現を使用しているため問題なし。

### 3. 「日常会話に必要な語彙数は約2,000語」
**判定: 正確（研究支持あり）**
- Nation (2001), West (1953) の研究: 日常会話の約90%は最頻出2,000語でカバーされることが示されている。
- これはESL/EFL研究で広く受け入れられている知見。

### 4. 「パッシブ語彙とアクティブ語彙の区別」
**判定: 正確（言語学の標準概念）**
- 受容語彙（Receptive vocabulary）と産出語彙（Productive vocabulary）の区別は第二言語習得研究の標準的概念（Nation, 2001; Melka, 1997）。

### 5. 「再認→再生→自動化」の3段階
**判定: 正確（認知心理学・習得研究に基づく）**
- Recognition → Recall → Automaticity は記憶・習得の標準的段階モデル（Anderson's ACT model, 1983）。

## 総合評価
**PASS** — 語彙数の数値は公的資料と研究に基づいており正確。パッシブ/アクティブ語彙の概念・3段階モデルも学術的に妥当。
