# Fact Check — 2026591382102749239

## チェック対象
「英語センス不要論の根拠。反復量だけが英語力を決める4つの証拠。」

## 検証項目

| 主張 | 評価 | 根拠・注記 |
|---|---|---|
| 発音は才能が影響する | ✅ 概念的に正確 | 音韻知覚の個人差（phonological sensitivity）はSLA研究で確認されている（Carroll 1981, MLAT）。絶対音感のような極端な例はあるが、発音習得への影響は「学習容易性の差」に過ぎず「不可能」ではない |
| 文法・語彙・リスニング・会話は反復量が決める | ✅ SLA研究と整合 | Nation（2001）の語彙習得研究では語との「出会い回数」（encounters）が定着を決定することを示す。文法の内在化も繰り返し使用が必要（Ellis 1994）。会話流暢性はDeKeyser（2007）の自動化理論と整合 |
| 「才能が効くのは発音のみ」という主張 | ⚠️ 簡略化あり | Language Aptitude（言語適性）は発音以外にも影響する可能性がある（Carroll 1981のMLAT：文法感度・記憶力等）。ただし元ポストも記事も「絶対的な主張」ではなく「最大の要因は反復量」という趣旨であり、教育的文脈での簡略化として許容範囲内 |
| ピアノと英語の習得曲線が同じ形 | ✅ 比喩として適切 | スキル習得の対数的成長曲線はAndersson（1982）のACTモデル等で確認されている。ピアノも英語も「練習量 → 習得」の曲線が類似する。図解に「概念図」と明示しているため問題なし |
| 「才能がないから話せない」は思い込み | ✅ 教育的に正確 | 固定マインドセット（Dweck 2006, Mindset）の代表的誤信念。成長マインドセットの観点から否定することは研究的根拠がある |

## 判定

**PASS（条件付き）**

条件：図解内の数値（才能5%・反復95%等）は概念的な説明図であり、特定研究の実測値ではないことを図解キャプション等で補足すること（図解内に既に注記済み）。

「才能が効くのは発音のみ」はやや簡略化だが教育的文脈での説明として許容範囲内。ハルシネーション・誇大主張なし。

## 注意点
- 言語適性（Language Aptitude）は複数要素からなるが、記事の趣旨（反復量の重要性）と矛盾しない
- ピアノの習得曲線図は「概念転用図」であり、実験データではないことが明示されている
