英語の比較級・最上級の作り方【-er/-est・more/most・不規則変化と例文】

  4. 不規則変化（good・bad・many・far など）
  5. as 〜 as 構文（同等比較）

「bigger」「more important」「best」——英語の比較表現は形容詞・副詞の語数によって形が変わり、さらに不規則変化もあるため混乱しやすい分野です。本記事では比較級・最上級の作り方ルールを網羅的に整理し、実際の使い方を例文とともに解説します。

比較級・最上級の基本：-er / -est をつける形

1音節の形容詞・副詞は、原則として語尾に -er（比較級） / -est（最上級） をつけます。

原級比較級最上級意味
talltallertallest背が高い
fastfasterfastest速い
oldolderoldest年老いた・古い
youngyoungeryoungest若い
cheapcheapercheapest安い

My sister is taller than me.

私の姉は私より背が高いです。（比較級 + than）

This is the fastest train in the country.

これはこの国で最も速い電車です。（最上級 → the + -est）

最上級には必ず the をつけるのが基本ルールです。ただし副詞の最上級では the を省略することもあります。

more / most を使う形（2音節以上の形容詞）

3音節以上の形容詞、および多くの 2音節の形容詞（特に -ful / -less / -ous / -ive / -ing / -ed で終わるもの）は more（比較級） / most（最上級） を前に置きます。

原級比較級最上級
beautifulmore beautifulmost beautiful
importantmore importantmost important
difficultmore difficultmost difficult
interestingmore interestingmost interesting
carefulmore carefulmost careful

Health is more important than wealth.

健康は富よりも大切です。

This is the most interesting book I've read this year.

これは今年読んだ中で最も興味深い本です。

2音節形容詞の扱い（-er か more か）

2音節の形容詞は -er / -est と more / most のどちらを使うか迷いがちですが、次のように覚えると便利です。

  • -y で終わる（happy, easy, heavy 等）→ -ier / -iest（y → i に変化）
  • -er / -ow / -le で終わる（clever, narrow, simple 等）→ -er / -est が自然（more も使用可）
  • その他の2音節語（modern, famous, common 等）→ more / most

Today's test was easier than yesterday's.

今日のテストは昨日のより簡単でした。（easy → easier）

スペリングの変化ルール

-er / -est を付ける際のスペリング変化には規則があります。

条件変化ルール例
eで終わる語-r / -st をつけるlarge → larger / largest
子音字+y で終わる語y を i に変えて -er / -esthappy → happier / happiest
短母音+子音字1つで終わる語子音字を重ねて -er / -estbig → bigger / biggest
上記以外そのまま -er / -est をつけるtall → taller / tallest

This bag is bigger than I expected.

このバッグは思っていたより大きいです。（big → bigger：子音を重ねる）

She is the nicest person I've ever met.

彼女は私が会った中で最も親切な人です。（nice → nicest：e に st をつける）

不規則変化（good・bad・many・far など）

次の形容詞・副詞は -er / -est / more / most ではなく、不規則な形に変化します。これらはそのまま暗記するのが効率的です。

原級比較級最上級意味
good / wellbetterbestよい・上手に
bad / badlyworseworst悪い・ひどく
many / muchmoremost多い・多く
littlelessleast少ない
farfarther / furtherfarthest / furthest遠い（farther = 距離、further = 程度）

Your English is getting better every day.

あなたの英語は日々上手くなっています。

This is the worst weather we've had this year.

これは今年最も悪い天気です。

I need more time to finish this project.

このプロジェクトを終わらせるにはもっと時間が必要です。（many/much → more）

We need to discuss this further.

これについてさらに議論する必要があります。（further = 程度・段階が進む）

as 〜 as 構文（同等比較）

as + 原級 + as で「〜と同じくらい…」という同等比較を表します。2つのものが同程度であることを示す重要な構文です。

This hotel is as expensive as the one we stayed at last time.

このホテルは前回泊まったところと同じくらい高いです。

Please reply as soon as possible.

できるだけ早く返信してください。（ASAP = as soon as possible の略語）

否定形 not as 〜 as（劣等比較）

not as + 原級 + as で「〜ほど…ではない」という意味になります。

The new version is not as fast as the old one.

新バージョンは旧バージョンほど速くありません。

発展表現（the + 比較級・比較級の強調など）

the + 比較級, the + 比較級（〜すればするほど）

「The + 比較級 + S + V, the + 比較級 + S + V」で「〜すればするほど、ますます…」という意味を表します。

The more you practice, the better you get.

練習すればするほど、上達します。

The earlier you start, the more time you'll have.

早く始めるほど、時間が多くなります。

比較級の強調（much / far / a lot / even / a little + 比較級）

比較級の前に強調語を置くと、差の大きさを表現できます。

This route is much faster than the highway.

このルートは高速道路よりずっと速いです。（much で強調）

She speaks English far better than she used to.

彼女の英語は以前よりはるかに上手くなりました。（far で強調）

Today is a little warmer than yesterday.

今日は昨日より少し暖かいです。（a little で弱い差を表現）

最上級の強調（by far the + 最上級）

最上級に by far を加えると「断然〜」という強調になります。

She is by far the most talented player on the team.

彼女はチームの中で断然最も才能のある選手です。

まとめ

比較表現の要点をまとめます。

  • 1音節の語は -er / -est（tall → taller → tallest）
  • 3音節以上・多くの2音節語は more / most（important → more important → most important）
  • -y で終わる2音節語は y → i に変えて -ier / -iest（happy → happier → happiest）
  • 不規則変化はそのまま暗記（good → better → best / bad → worse → worst / many → more → most）
  • 同等比較は as + 原級 + as（not as … as = 劣等比較）
  • 最上級には the をつける（副詞の場合は省略可）
  • The + 比較級, the + 比較級 =「〜すればするほど」
  • much / far / a lot で比較級を強調できる
