英語の仮定法（if文）の使い方【仮定法過去・過去完了・would/could の例文解説】

英語の文法の中でも「難しい」と感じる人が多い仮定法。「もし〜だったら」という反事実の仮定を表す表現で、ビジネス英語や日常会話でも頻繁に使われます。

この記事では、仮定法過去・仮定法過去完了の形と使い方、would / could / might の使い分け、そして if 以外の仮定法表現まで、例文を交えてわかりやすく解説します。

直説法と仮定法の違い（事実の話 vs 反事実の想像）

まず「直説法」と「仮定法」の根本的な違いを押さえましょう。

  • 直説法：現実に起きていること・起きたことを述べる（If it rains, I'll stay home.）
  • 仮定法：現実とは反対の仮定・空想・願望を述べる（If it rained, I would stay home.）
If I have time, I will help you. （直説法 — 本当に時間があるかもしれない）
時間があれば、手伝います。（実際にそうなる可能性がある）
If I had time, I would help you. （仮定法過去 — 実際には時間がない）
もし時間があれば（でも実際にはない）、手伝うのに。

仮定法では、動詞を1段階「過去」にずらすことで「現実ではない」というシグナルを出します。これが仮定法の核心的なルールです。

仮定法過去の形と使い方（現在の事実に反する仮定）

仮定法過去は「今の現実に反する仮定」を表します。英語では現在の反事実を表すために過去形を使うのが特徴です。

仮定法過去の形
If + 主語 + 動詞の過去形 〜, 主語 + would / could / might + 動詞の原形 〜.

be動詞の場合、主語が I / he / she などの単数形でも were を使うのが文法的に正確とされています（会話では was も使われます）。

If I were a bird, I could fly anywhere.
もし私が鳥だったら、どこへでも飛んでいけるのに。（実際は人間）
If I had more money, I would travel around the world.
もっとお金があれば、世界中を旅するのに。（実際にはそれほどお金がない）
If she spoke English fluently, she could get that job.
彼女が英語を流暢に話せれば、その仕事に就けるのに。（実際には話せない）
「If I were you」は頻出表現

"If I were you, I would ..." は「もし私があなたなら〜するのに」という助言の定型表現です。会話でよく使うので、フレーズごと覚えておきましょう。

If I were you, I would apologize immediately.
もし私があなたなら、すぐに謝ります。

仮定法過去完了の形と使い方（過去の事実に反する仮定）

仮定法過去完了は「過去の現実に反する仮定」を表します。「あのときこうだったら……」という後悔・反省を語るときに使います。

仮定法過去完了の形
If + 主語 + had + 過去分詞 〜, 主語 + would / could / might + have + 過去分詞 〜.
If I had studied harder, I would have passed the exam.
もっと一生懸命勉強していたら、試験に合格できていたのに。（実際は落ちた）
If she had left earlier, she would have caught the train.
もし彼女がもっと早く出発していたら、電車に乗れていたのに。（実際は乗り遅れた）
If we had known about the problem, we could have fixed it.
その問題を知っていたなら、修正できていたのに。（実際は知らなかった）

仮定法過去 vs 仮定法過去完了の比較

種類扱う時制典型例
仮定法過去現在の反事実If I were rich, I would ...
仮定法過去完了過去の反事実If I had been rich, I would have ...

would / could / might の使い分け

仮定法の帰結節では would / could / might のいずれかを使います。どれを選ぶかで意味のニュアンスが変わります。

would：ほぼ確実な帰結

「〜するだろう / 〜するのに」という、最も一般的に使われる形です。

If I won the lottery, I would buy a house.
宝くじに当たったら、家を買うのに。

could：能力・可能性

「〜できるのに / 〜できたのに」という意味を添えます。

If I had more time, I could learn a new language.
もっと時間があれば、新しい言語を学べるのに。

might：弱い可能性・不確実な帰結

「もしかしたら〜かもしれないのに」という、不確かなニュアンスを表します。

If you asked her nicely, she might agree.
丁寧にお願いすれば、彼女は同意するかもしれないのに。
選び方のポイント

「〜するだろう（ほぼ確実）」→ would、「〜できる（能力・可能性）」→ could、「〜かもしれない（弱い可能性）」→ might。まずは would を基本として覚え、能力や可能性を表したいときに could を選ぶ練習から始めましょう。

if以外の仮定法表現（I wish / as if / without）

仮定法は if 節だけに使われるわけではありません。以下の表現でも同じ「反事実」の意味を表せます。

I wish + 仮定法過去（現在の願望）

「〜だったらいいのに」という現在の不満・願望を表します。

I wish I could speak French.
フランス語が話せたらいいのに。（実際には話せない）
I wish I were taller.
もっと背が高ければいいのに。

I wish + 仮定法過去完了（過去への後悔）

「〜だったらよかったのに」という過去への後悔を表します。

I wish I had taken that opportunity.
あのチャンスを掴んでいたらよかったのに。

as if / as though（まるで〜であるかのように）

実際はそうでないのに、そうであるかのように振る舞う様子を描写します。

He talks as if he knew everything.
彼はまるで何でも知っているかのように話す。（実際は知らない）
She acted as if nothing had happened.
彼女はまるで何も起きなかったかのように振る舞った。

without / but for（〜がなければ）

if 節の代わりに「〜がなければ」という条件を表します。

Without your help, I couldn't have finished the project.
あなたの助けがなければ、そのプロジェクトを終えられなかった。
But for the heavy rain, the game would have continued.
大雨がなければ、試合は続いていただろう。

まとめ

仮定法のポイントをまとめると次のようになります。

  • 仮定法の核心：動詞を1段階過去にずらすことで「現実ではない」ことを示す
  • 仮定法過去：If + 過去形 〜, would/could/might + 原形 〜（現在の反事実）
  • 仮定法過去完了：If + had + 過去分詞 〜, would/could/might + have + 過去分詞 〜（過去の反事実）
  • would/could/might：確実性の高い帰結は would、能力は could、弱い可能性は might
  • if 以外の仮定法：I wish、as if、without なども同じルールで仮定法を使う

仮定法は「時制のずれ」というルールを一度理解すれば、形を予測できるようになります。定番の例文を声に出して繰り返し練習し、文法クイズでアウトプットしてみましょう。
