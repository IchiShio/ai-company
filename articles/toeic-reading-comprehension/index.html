TOEIC Part7読解問題の解き方【スキャニング・パラグラフ読みで時間内に解く方法2026】

本記事はアフィリエイト広告を含みます。

TOEICのリーディングセクション（Part 5〜7）は100問・75分で解く必要があります。その中でもPart7は最も問題数が多く、時間配分を誤ると後半の問題を解き終えられないという失点リスクがあります。本記事では、スキャニングとパラグラフ読みを組み合わせた効率的な読解法と、スコア帯別の取捨選択戦略を解説します。

Part7の基本情報と時間配分の考え方

TOEICは全200問・120分の試験で、10〜990点・5点刻みのスコアが出ます。リーディングセクション（Part 5〜7）の100問は75分で解きます。

パート問題形式問題数
Part 5短文穴埋め30問
Part 6長文穴埋め（4セット）16問
Part 7長文読解（シングル・ダブル・トリプル）54問

Part7の54問に使える時間の目安は約50〜55分です。1問あたり平均約1分以内で処理する計算になります。ただし設問数・文書の長さによって問題ごとの時間は異なるため、一律1分という考え方は実態に合いません。

推奨タイムマネジメント

パート目標時間補足
Part 5（30問）約10〜12分1問20〜24秒ペース
Part 6（16問）約8〜10分1セット2〜2.5分
Part 7（54問）約53〜57分残り時間を全投入

Part 5・Part 6を早く終わらせるほど、Part7に時間を使えます。Part 5は知識問題が多いため、1問10秒で解けるものと30秒かかるものを素早く判別して進める習慣をつけましょう。

スキャニングでキーワードを高速検索する

スキャニングとは、文書を最初から最後まで丁寧に読むのではなく、答えに関係するキーワードだけを素早く探す技法です。Part7の設問に答えるうえで最も実用的な読み方です。

スキャニングの手順

  1. 設問を先に読み、何を探すか（固有名詞・数値・日付・条件など）を頭に入れる
  2. 文書全体を上から下へ目を滑らせるように眺める（1語1語読まない）
  3. 設問のキーワードに近い語が目に入ったら、その周辺だけ精読する
  4. 答えを特定したら、次の設問のスキャニングに移る

スキャニングが効く設問タイプ

  • 「〜はいつですか」（日付・時間）
  • 「〜はいくらですか」（金額・割引率）
  • 「〜の連絡先は何ですか」（電話番号・メールアドレス）
  • 「〜に関して正しいものはどれか」（NOT問題以外の事実確認）

パラグラフ読みで全体構造を掴む

スキャニングだけでは答えられない問題があります。「この文書の目的は何か」「筆者が最も伝えたいことは何か」などの主題・目的・主張を問う問題は、文書全体の流れを理解する必要があります。

パラグラフ読みの基本ルール

各パラグラフの最初の1文（トピックセンテンス）を読むだけで、その段落の主旨がわかることが多くあります。全文を読まずに、各段落の先頭文だけ追うだけで文書の骨格が掴めます。

  • 第1段落の先頭文：文書の目的・背景
  • 第2〜中間段落の先頭文：詳細・根拠・条件
  • 最終段落の先頭文：まとめ・行動依頼・次のステップ

「NOT問題」の解き方

「Which of the following is NOT mentioned?」のような問題は、消去法が基本です。選択肢A〜Dを順番に文書で確認し、言及されているものを消去して最後に残ったものが正解です。この作業は時間がかかるため、他の問題を先に解いてから最後に取り組むのが効率的です。

設問タイプ別の解き方

目的・主旨問題

"What is the purpose of this email?" など。文書の冒頭（第1〜2段落）に目的が書かれていることが多いため、冒頭を重点的に読みましょう。

詳細情報問題

"According to the notice, when will the office close?" など。スキャニングで対象キーワード（"close" / "office"）を探し、周辺の数値・日付を読み取ります。

推測・意図問題

"What does the writer imply about the project?" など。答えは文書に明示されず、前後の文脈から推測する必要があります。パラフレーズ（言い換え）を含む選択肢が正解になりやすい傾向があります。

文挿入問題

文書中の [1] [2] [3] [4] のどこに1文を挿入するかを問う問題は、接続詞・代名詞・指示語（this / these / it）に注目して前後の文脈を確認します。

  • 挿入文に However / Therefore / In addition などがある → 前後の論理関係を確認
  • 挿入文に it / this / they がある → 直前に対応する名詞が必要

ダブル・トリプルパッセージの攻略法

Part7の後半にはダブルパッセージ（2文書）とトリプルパッセージ（3文書）が出ます。複数の文書にまたがる情報を統合して答える「クロス問題」が含まれるため、単一文書より時間がかかります。

クロス問題の見分け方

設問文に "in the second document" / "according to both notices" / "based on the email and the schedule" などの表現があれば、複数文書を参照する必要があるクロス問題です。それ以外の設問は1つの文書だけで解けるケースが多くあります。

ダブル・トリプルの解き方の流れ

  1. 設問5問を先に全部読む
  2. 第1文書を読みながら解ける問題をマークする
  3. 第2（・第3）文書を読みながら残りを解く
  4. クロス問題は最後にまとめて処理する

時間切れ防止のルール：トリプルパッセージは1セット5問に対して3つの文書を読む必要があるため、1セットに5〜7分かかることもあります。時間が足りなくなりそうな場合は、クロス問題だけ後回しにして他の問題を先に解き切ることを優先しましょう。

取捨選択戦略：捨て問の判断基準

現在のスコア帯によって、Part7での最適な取捨選択が異なります。全問を解こうとして時間切れになるより、優先度の高い問題を確実に正解するほうが高スコアにつながることがあります。

目標スコアPart7の戦略
〜500点シングルパッセージ優先。ダブル以降は目標問題数を決めて臨む
500〜700点シングル全問＋ダブル優先。トリプルは時間が余れば取り組む
700〜800点全問取り組む。時間切れリスクに備えてNOT問題を後回しにする
800点以上全問解き切ることを前提に、Part5・6の処理速度を上げてPart7に時間を確保する

「時間をかけすぎない問題」の判断基準

  • 30秒考えても答えが絞れない → とりあえずマークして次へ（戻れる時間があれば再考）
  • NOT問題（全選択肢を文書で確認する必要がある） → 後回しにする
  • クロス問題（複数文書を参照する） → 後回しにする

読解速度を上げるための語彙強化

Part7で時間切れになる根本原因の一つは語彙不足による読み詰まりです。意味のわからない単語が多い文を読むと、前後の文脈から類推しようとして大幅に時間を消費します。

Part7で頻出する語彙カテゴリ

  • ビジネス・取引系：invoice / procurement / vendor / compliance / subsidiary
  • 人事・採用系：résumé / compensation / probationary period / on-site interview
  • 施設・不動産系：renovation / tenant / premises / lease / utilities
  • 広告・マーケティング系：promotional offer / endorsement / demographic / campaign
  • 旅行・交通系：itinerary / boarding pass / layover / roundtrip fare

これらの語彙を公式問題集の文脈で覚えておくと、Part7の読解速度が上がる傾向があります。またGrammarUpの文法クイズで文構造への理解を深めることも、長文読解のスピードアップに役立ちます。
