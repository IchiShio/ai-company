# 公務員から外資ITに転職して年収2倍にしたテンプレート集 -- 職務経歴書・面接Q&A・LinkedIn・年収交渉

## はじめに -- このテンプレートパックの使い方

私は元国家公務員（霞が関）で、IT外資に転職して年収を2倍以上にした「ひつじ」です。

このテンプレートパックは、公務員から民間への転職に必要な「実用的なテンプレート」を、コピペしてそのまま使えるレベルで網羅したものです。

収録内容は以下の4つ:

**1. 職務経歴書テンプレート 拡張版**
- 5業界向けの職務経歴書テンプレート（IT/コンサル/外資IT/スタートアップ/メーカー商社）
- 全5パターンのカバーレター
- 全5パターンの英文レジュメ
- 公務員スキル翻訳辞書（55項目）
- 自己PR例文10パターン

**2. 面接想定Q&A 50選（STAR法回答例付き）**
- 一般質問15問
- 公務員特有の質問15問
- 外資特有の質問10問（英語面接含む）
- 圧迫面接・想定外質問への対処法10問

**3. LinkedIn完全最適化ガイド**
- プロフィール全セクションの書き方
- 公務員経験の英語記述テンプレート
- Headline例10パターン / About例5パターン
- 直接応募の具体的手順
- つながりリクエストのメッセージテンプレート

**4. 年収交渉テンプレート集**
- オファーレターの読み方チェックリスト
- 年収交渉メール5パターン（日本語3本・英語2本）
- 口頭での交渉スクリプト3パターン
- 複数オファーの比較テンプレート
- 交渉NG表現リスト

### 使い方

1. まず自分の志望業界に合ったテンプレートを選ぶ
2. 「○○」「XX」の部分を自分の経歴に書き換える
3. 必ず自分の言葉に直してから使う（コピペのまま提出するのはNG）
4. 面接Q&Aは回答例を参考に、自分のエピソードを当てはめる
5. 年収交渉は内定後に使う（選考中は使わない）

最も大切なのは「テンプレートをそのまま使うこと」ではなく「テンプレートを参考に、自分だけの武器を作ること」。公務員経験を民間の言葉に翻訳するための辞書として活用してほしい。

## パターン4: 公務員 → スタートアップ向け

### 職務要約

○○省にて約XX年間、政策立案・法令改正・予算編成等の行政実務に従事。総括補佐として省内横断プロジェクトを複数統括し、ゼロベースでの制度設計や限られたリソースでの成果創出を経験してきた。行政内部での「0→1」の政策立ち上げ経験を活かし、スタートアップ環境での事業開発・組織立ち上げに貢献したい。

### 職務経歴

#### ○○省 ○○局 ○○課（20XX年X月〜20XX年X月）

**役職**: 総括補佐（課長補佐級）

**業務内容**:
- 新規政策の企画立案（ゼロベースでの制度設計からローンチまで）
  - 民間翻訳: 「新規事業・プロダクトの企画から立ち上げまでを一貫して推進」
- 限られた人員（担当者X名）での大規模プロジェクト遂行
  - 民間翻訳: 「少数精鋭チームでの高インパクトプロジェクト推進（リソース制約下での優先順位判断）」
- 外部有識者・民間企業とのヒアリング・共同検討（XX社以上）
  - 民間翻訳: 「業界パートナーとの共同事業開発・プロダクトフィードバック収集」
- 省内で前例のない取り組みの企画・社内説得・実行
  - 民間翻訳: 「既存組織内での新規施策の社内起業的な立ち上げ（イントレプレナー経験）」
- 国会対応・メディア対応を含む対外コミュニケーション全般
  - 民間翻訳: 「IR・PR・ステークホルダーコミュニケーション」

#### ○○省 ○○局 ○○課（20XX年X月〜20XX年X月）

**役職**: 係長

**業務内容**:
- ○○分野の実態調査・データ分析・政策提言
  - 民間翻訳: 「市場調査・ユーザーリサーチに基づく事業仮説の検証と戦略提言」
- 複数省庁にまたがる連携プロジェクトの実務担当
  - 民間翻訳: 「他部門・他社との連携プロジェクトのハンズオン実行」
- 後輩X名の指導・業務アサイン
  - 民間翻訳: 「チームビルディングとメンバー育成」

### スキル・資格

・資格 → TOEIC XX / 英検○級
・PCスキル → Excel（ピボットテーブル・マクロ）、PowerPoint、Word
・特筆スキル → ゼロベースの企画立案・制度設計、ステークホルダー調整、少数チーム運営
・語学 → 英語（ビジネスレベル/学習中の場合はその旨記載）


### 自己PR

**ゼロイチの立ち上げ経験**: 省内で前例のなかった○○施策を、企画段階から実行まで主導した。上層部への説明・予算確保・関係者の巻き込みを一人で行い、XX年度に正式に制度化された。「誰もやったことがないことを形にする力」はスタートアップでの事業立ち上げに直結すると考えている。

**リソース制約下での成果創出**: 公務員は潤沢なリソースがあるイメージを持たれがちだが、実際には限られた人員と予算の中で最大の成果を出すことが常に求められる。担当者X名で年間予算XX億円規模の事業を回してきた経験は、スタートアップの「少ない資源で大きなインパクトを出す」文化と親和性が高い。

**行政知見を活かしたGR（Government Relations）**: スタートアップにとって規制対応や官公庁との関係構築は重要な経営課題。行政の意思決定プロセス・キーパーソンの特定方法・効果的なアプローチ手法を実務レベルで熟知しているため、GR機能の立ち上げに即戦力として貢献できる。

## 自己PR例文10パターン（業種 x 強み別）

### 1. IT企業 x プロジェクトマネジメント力

私の最大の強みは、複数のステークホルダーが関わるプロジェクトを期限内に確実に着地させるマネジメント力です。総括補佐として、省内5部署以上・外部機関XX団体が関わる横断プロジェクトを複数統括してまいりました。各組織の利害が異なる中で、個別の要望をヒアリングし、全体最適となる落としどころを設計して合意を形成するプロセスを繰り返してきました。IT業界のプロダクト開発においても、開発・営業・マーケティング・カスタマーサクセスなど多くの部門が関わります。異なる利害を持つチーム間の橋渡し役として、プロジェクトを前に進める力を発揮できると考えています。

### 2. IT企業 x ドキュメンテーション力

法令文書・政策資料・国会答弁書・プレゼン資料など、目的と読み手に応じた文書を日常的に大量に作成してきました。特に国会対応では、複雑な政策を「非専門家に正確に、かつ短時間で理解できるように」伝える技術が求められます。この「情報の構造化」と「読み手に合わせた表現の最適化」は、IT企業における仕様書・企画書・社内Wiki・顧客向け提案書の作成にそのまま活かせるスキルです。入社後は、チーム全体のドキュメント品質を底上げする役割も担いたいと考えています。

### 3. コンサルファーム x 構造化思考

政策立案のプロセスでは、(1)現状分析 → (2)課題の構造化 → (3)オプション策定 → (4)評価基準に基づく選定 → (5)実行計画の策定、というフレームワークを毎日のように使ってきました。例えば○○制度の改正では、XX件のデータと利害関係者ヒアリングを統合し、3つの改革オプションを費用対効果と実現可能性の二軸で評価して、最終的に関係省庁XX省の合意を得ました。この構造化された問題解決アプローチはコンサルティングの基本と本質的に同じであり、入社初日からプロジェクトに貢献できると確信しています。

### 4. コンサルファーム x パブリックセクター知見

行政の内部で意思決定がどのように行われるか、予算がどのサイクルで動くか、キーパーソンは誰か。これらは外から調べても分からない「暗黙知」です。私はXX年間の行政経験を通じて、この暗黙知を実務レベルで蓄積してきました。コンサルティングファームのパブリックセクター案件においては、クライアントの内部力学を理解した上で実行可能な提案ができることが、他候補者にはない明確な差別化要因になると考えています。

### 5. 外資IT x グローバル対応力

TOEIC XX・英検○級の英語力を有し、海外の行政機関との情報交換や国際会議への参加を経験してきました。外資IT企業の日本オフィスでは日本語が主な業務言語ですが、本社との連携やグローバルプロジェクトへの参画場面では英語力が差別化要因になります。加えて、異文化環境への適応力にも自信があります。公務員として省庁間の「異なるカルチャー」を横断してきた経験は、グローバル企業の多様な組織文化に適応する力と通じるものがあると感じています。

### 6. 外資IT x パブリックセクター営業適性

外資IT企業にとって、官公庁・自治体は重要かつ複雑なクライアントセグメントです。行政の調達プロセス（仕様書作成→入札→評価→契約→検収）を熟知し、意思決定のキーパーソンや根回しの作法を理解していることは、パブリックセクター営業・アカウントマネジメントにおいて大きなアドバンテージです。私はXX年間、この「行政の内側」で働いてきました。その経験を買い手側から売り手側に活かすことで、貴社のパブリックセクタービジネスの拡大に貢献したいと考えています。

### 7. スタートアップ x ゼロイチ立ち上げ力

省内で前例のなかった○○施策を、構想段階から制度化まで一人で推進した経験があります。上層部を説得するためのロジック構築、予算の確保、他部署の巻き込み、反対意見への対処。これらすべてを、既存のマニュアルや前例なしに遂行しました。スタートアップにおける「誰もやったことがないことを形にする」プロセスと本質的に同じです。加えて、リソースが限られた環境での意思決定には慣れています。担当者X名で年間XX億円の事業を動かしてきた経験は、スタートアップの「少ない資源で大きなインパクトを出す」文化と高い親和性があります。

### 8. スタートアップ x GR（Government Relations）

スタートアップにとって、規制対応は事業の命運を左右する重要課題です。しかし多くのスタートアップにはGR機能が存在しません。私は行政の意思決定プロセスを内部から熟知しており、「どの部署に」「どのタイミングで」「どのようなアプローチで」コンタクトすれば効果的かを実務レベルで把握しています。規制緩和の相談、官公庁との実証実験の設計、公共調達への参入支援など、GR機能をゼロから立ち上げ、事業成長のアクセルにすることができます。

### 9. メーカー/商社 x 業界知見とネットワーク

○○省にてXX年間、○○産業に関する政策立案に従事する中で、業界の主要企業・業界団体・研究機関と幅広いネットワークを構築しました。単なる名刺交換ではなく、政策の共同検討やヒアリングを通じた信頼関係に基づくネットワークです。メーカー/商社においては、新規事業開発、アライアンス構築、市場調査の場面でこのネットワークを即座に活用できます。さらに、産業政策の方向性を読む力は、中長期の事業戦略策定において他候補者にはない価値を提供できると考えています。

### 10. メーカー/商社 x 規制対応と事業戦略の統合

○○産業の振興と規制の両方を担当した経験から、「規制は制約」ではなく「正しく理解すれば事業機会」であることを実感しています。新しい規制が導入される際、多くの企業はコンプライアンス対応に追われますが、事前に規制の方向性を読み、先手を打てる企業が市場で優位に立ちます。私は規制の設計側にいた人間として、「次にどのような規制が来るか」「その規制が事業にどう影響するか」を先読みする力を持っています。この力は、メーカー/商社の事業企画・経営企画において、競合他社に対する明確な差別化になります。

# 第2部: 面接想定Q&A 50選（STAR法回答例付き）

公務員から民間への転職面接で頻出する質問を、4カテゴリ50問で網羅した。各質問に「質問の意図」「回答のポイント」「STAR法回答例」を付けている。回答例はあくまで骨格であり、自分の具体的なエピソードに差し替えて使ってほしい。

STAR法とは:
- **S**ituation（状況）: どのような場面だったか
- **T**ask（課題）: 何を求められていたか
- **A**ction（行動）: 具体的に何をしたか
- **R**esult（結果）: どのような成果が出たか

---

## カテゴリ1: 一般質問（15問）

### Q1. 自己紹介をお願いします。

**質問の意図**: 論理的に自分を要約できるか。話の構成力・コミュニケーション力を見ている。

**回答のポイント**:
- 2分以内にまとめる（長すぎは減点）
- 「経歴 → 強み → 志望動機」の3部構成

「はい、採用します。理由は、XX年間のPM力がこのポジションの即戦力になること、パブリックセクター知見は社内で育てにくいスキルであること、覚悟を持った転職であること。業界知識のキャッチアップは必要ですが、3〜6ヶ月で解決可能な課題であり、長期的な価値と比較すれば採用すべきと判断します。」

---

### Q48. 転職回数ゼロですが、環境変化に弱いのではないですか。

**回答例**:
「公務員は2〜3年ごとに異なる部署に異動するため、実質的にXX回の『社内転職』を経験しています。毎回新しい分野をゼロから学び、新しい上司・同僚と関係を構築してきました。環境変化への適応力は、むしろ公務員キャリアで最も鍛えられたスキルです。」

---

### Q49. 今日の面接のパフォーマンスを自己採点するなら何点ですか。

**回答例**:
「80点です。良かった点は経験を民間の言葉に翻訳して伝えられたこと。残り20点は、○○の質問でもう少し簡潔に回答できたはずという反省。次の機会をいただければ改善します。」

---

### Q50. 最後に、あなたを採用すべき理由を30秒で述べてください。

**回答例（30秒）**:
「理由は3つ。第一に、XX年間・XX部署以上のプロジェクト統括の実行力。第二に、行政の意思決定プロセスを内側から知る市場で希少な人材であること。第三に、国会対応で鍛えたプレッシャー下での即日アウトプット力。業界知識は入社後に習得しますが、この3つの力は後天的に身につけるのが最も困難なスキルです。初年度から成果を出す自信があります。」

## 5. 交渉で使ってはいけないNG表現リスト

・「この条件では辞退します」
  理由 → 最後通牒は関係を壊す
  代替表現 → 「ご検討いただけると嬉しいです」
・「他社はもっと出してくれます」
  理由 → 圧力をかけるのはNG
  代替表現 → 「他社からもオファーをいただいており、総合的に判断したい」
・「生活が苦しいので」
  理由 → 個人的事情は交渉材料にならない
  代替表現 → 「市場相場と私のスキルを考慮すると」
・「公務員を辞めてまで来るのだから」
  理由 → 自己犠牲アピールは逆効果
  代替表現 → 「御社での成長機会に期待して」
・「とりあえず上げてほしい」
  理由 → 根拠がない
  代替表現 → 具体的な金額と理由を提示
・「他社から○○万円のオファーが」（嘘）
  理由 → バレたら内定取消
  代替表現 → 事実のみ伝える
