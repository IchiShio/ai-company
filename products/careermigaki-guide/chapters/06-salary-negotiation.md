# 第6章: 年収交渉 — SOとボーナスの読み方

## 外資ITの報酬は「基本給」だけではない

公務員の給与体系はシンプルだ。俸給表に基づく基本給に、各種手当が加わる。ボーナスも支給月数がほぼ決まっている。予測可能で安定している一方、交渉の余地はほぼゼロだ。

外資ITの報酬体系はこれとまったく異なる。基本給はあくまで報酬の一部であり、ボーナス、ストックオプション（SO）やRSU（制限付き株式ユニット）が加わって「トータルコンペンセーション（総報酬）」を構成する。

私の場合、IT外資初年度の報酬は額面1,300万円後半に、ボーナス12.5%、そしてストックオプション（SO）の付与があった。公務員時代の年収650〜750万円と比較すると、基本給だけでも大幅な増加だが、ボーナスとSOを含めた総報酬はさらに大きな数字になる。

この章では、オファーレターの読み方、SOとRSUの仕組み、そして交渉のポイントを解説する。


## オファーレターの構成要素

外資ITからオファーレターを受け取ると、以下のような項目が記載されている。それぞれの意味を正しく理解することが、交渉の第一歩だ。

### 1. Base Salary（基本給）

年間の基本給。月額ではなく年額で提示されることが多い。これが毎月の給与として支払われる確定報酬だ。

### 2. Target Bonus / Variable Pay（目標ボーナス）

基本給に対する一定割合で設定されるボーナス。私の場合は基本給の12.5%だった。

注意すべきは「Target（目標）」という言葉だ。これは「100%達成した場合に支払われる金額」であり、業績や個人評価によって上下する。会社の業績が好調であれば目標を超えることもあるし、不調であれば下回ることもある。

公務員のボーナスとの最大の違いは、この変動性にある。ただし、外資ITの多くの企業では、目標ボーナスの達成率が極端に低くなることは稀だ。

### 3. Stock Option（SO）/ RSU

これが外資IT報酬の最大の特徴であり、多くの人が理解に苦しむ部分でもある。詳しくは次のセクションで解説する。

### 4. Sign-on Bonus（サインオンボーナス）

入社時に一度だけ支払われるボーナス。前職の退職に伴って失う報酬（未消化のボーナスやSOなど）を補填する目的で提示されることが多い。全てのオファーに含まれるわけではない。

### 5. Benefits（福利厚生）

健康保険、確定拠出年金、有給休暇日数、リモートワーク制度など。日本の外資ITでは、社会保険は日本の制度に準拠しているため、公務員時代と大きく変わらない部分も多い。


## ストックオプション（SO）の仕組み

SOは「将来、あらかじめ決められた価格（行使価格）で自社株を購入できる権利」だ。株価が行使価格を上回っていれば、その差額が利益になる。

### SOのライフサイクル

**1. 付与（Grant）**
入社時にSOが付与される。「1,000株分のSOを付与」のように株数で提示される。この時点では、まだ株を購入する権利は発生していない。

**2. ベスティング（Vesting）**
付与されたSOが実際に行使可能になるプロセス。多くの企業では4年間のベスティングスケジュールを採用しており、一般的なパターンは以下の通りだ。

- 1年目: 25%がベスト（クリフ期間）
- 2年目以降: 残り75%が毎月または四半期ごとに均等にベスト

「クリフ」とは、最初の1年間は一切ベストされず、1年経過した時点で25%が一括でベストされる仕組みだ。これは企業側が短期離職のリスクを軽減するために設けている。

**3. 行使（Exercise）**
ベスト済みのSOを使って、行使価格で株を購入する。株価が行使価格より高ければ、差額が利益になる。

**4. 売却（Sell）**
購入した株を市場で売却し、利益を確定する。上場企業であれば市場で売却できるが、未上場企業の場合は売却の機会が限られる。

### SOの注意点

- 行使価格より株価が下がった場合、SOは「アンダーウォーター」となり、行使しても損をするため実質的に無価値になる
- 行使時と売却時にそれぞれ税金が発生する
- 退職時には通常、ベスト済みのSOを一定期間内（多くの場合90日以内）に行使する必要がある
- 未上場企業のSOは流動性が低く、いつ現金化できるか不確実だ


## RSU（Restricted Stock Unit）との違い

大手外資ITでは、SOではなくRSU（制限付き株式ユニット）を採用しているケースが多い。

RSUは「ベスティング期間を経て、自動的に株式が付与される」仕組みだ。SOとの最大の違いは、RSUには行使価格がないこと。ベスト時点の株価がそのまま報酬の価値になる。

**SOとRSUの比較:**

| 項目 | SO | RSU |
|---|---|---|
| 付与されるもの | 株を買う「権利」 | 株そのもの |
| 行使価格 | あり | なし |
| 株価下落時 | 無価値になりうる | 価値は下がるがゼロにはならない |
| リスク | 高い（ハイリスク・ハイリターン） | 低い（確実性が高い） |
| 主な採用企業 | スタートアップ・成長企業 | 大手IT企業 |

RSUのほうがリスクが低い分、確実に報酬として計算しやすい。一方、SOはスタートアップが上場や急成長した場合に大きなリターンを得られる可能性がある。


## 交渉すべきポイント

オファーレターを受け取ったら、交渉のフェーズに入る。外資ITでは、オファーの交渉は一般的な行為であり、交渉したこと自体がマイナス評価になることはない。

### 交渉の優先順位

**1. 基本給（Base Salary）**
最も交渉しやすく、かつインパクトが大きい。基本給が上がれば、それに連動するボーナス（基本給の12.5%など）も自動的に上がる。翌年以降の昇給のベースにもなるため、入社時に少しでも高い基本給を確保しておくことの効果は累積する。

**2. サインオンボーナス（Sign-on Bonus）**
前職で失う報酬がある場合、それを根拠にサインオンボーナスを要求できる。一度きりの支払いであるため、企業側も比較的応じやすい項目だ。

**3. SO / RSU**
株式報酬の追加付与を交渉することも可能だ。ただし、SOやRSUの付与数は職位や社内グレードに紐づいている場合が多く、基本給ほどの柔軟性がないこともある。

### やってはいけない交渉NG例

**NG 1: 根拠なく「もっとほしい」と言う**
「他社ではもっともらえる」「自分の経験を考えればもっと高いはずだ」といった曖昧な主張は、交渉ではなく要求になる。交渉は、具体的なデータや根拠に基づいて行うものだ。

**NG 2: 最初のオファーに即座にYesと言う**
交渉の余地があるにもかかわらず、最初のオファーをそのまま受け入れるのはもったいない。「検討する時間をいただけますか」と言うだけでも、交渉の余地が生まれる。

**NG 3: 嘘をつく**
「他社からもっと高いオファーが出ている」と嘘をつくのは、絶対にやめるべきだ。外資ITの業界は想像以上に狭く、採用担当者同士のネットワークもある。嘘が発覚すれば、オファー取り消しだけでなく、業界内での信用を失うリスクがある。


## 複数オファーの比較方法

複数の企業からオファーが出た場合、単純に基本給だけで比較するのは危険だ。以下の観点でトータルに比較する必要がある。

### 比較すべき項目

**1. トータルコンペンセーション（年間総報酬）**
基本給 + ボーナス（目標値）+ SO/RSUの年間ベスト分を合算して比較する。SOやRSUについては、現在の株価で計算した参考値を使う。

**2. ベスティングスケジュール**
4年均等か、フロントローディング（初年度に多く付与）か、バックローディング（後半に多く付与）かで、初年度の実質報酬が大きく変わる。

**3. 株式の流動性**
上場企業の株であれば市場で売却できるが、未上場企業の場合はいつ現金化できるか不確実だ。未上場企業のSOは、割り引いて評価する必要がある。

**4. キャリアへのインパクト**
報酬だけでなく、そのポジションで得られるスキル、経験、キャリアパスも重要な比較軸だ。短期的に報酬が高くても、キャリアの幅が狭まるポジションは慎重に検討すべきだ。

**5. 福利厚生と働き方**
リモートワークの可否、有給休暇日数、健康保険の内容なども比較する。これらは数字に換算しにくいが、日常の満足度に直結する。


## オファーレターを受け取ったらやること

1. **焦って返事しない。** 最低でも数日の検討時間を確保する
2. **トータルコンペンセーションを計算する。** 基本給だけでなく、ボーナス、SO/RSUを含めた年間総報酬を算出する
3. **交渉ポイントを整理する。** 根拠とともに、何をどれだけ上げたいかを明確にする
4. **交渉は書面（メール）で行う。** 口頭だけだと認識の齟齬が生まれやすい。メールで要望を伝え、回答もメールでもらう
5. **最終的なオファーを書面で確認する。** 交渉結果が反映された最終版のオファーレターを必ず受け取る


## この章のまとめ

- 外資ITの報酬は基本給+ボーナス+SO/RSUの「トータルコンペンセーション」で評価する
- SOは「株を買う権利」、RSUは「株そのもの」。リスクとリターンの構造が異なる
- ベスティング（4年が一般的）とクリフ（1年目の待機期間）の仕組みを理解しておく
- 交渉は外資では一般的な行為。基本給を最優先に、根拠を持って交渉する
- 複数オファーは基本給だけでなく、トータルコンペンセーション、ベスティング、キャリアインパクトを含めて比較する
- 嘘をつく、根拠なく要求する、即座にYesと言う、の3つは交渉のNG行動
