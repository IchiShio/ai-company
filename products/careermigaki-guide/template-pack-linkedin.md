# LinkedIn完全最適化ガイド

公務員から民間への転職において、LinkedInは最重要のプラットフォーム。特に外資IT企業への応募では、LinkedInプロフィールが「もう一つの職務経歴書」として機能する。このガイドでは、公務員経験をLinkedIn上で最大限に活かすための具体的なテンプレートと手順を網羅する。

---

## 1. LinkedInプロフィール各セクションの書き方

### 1-1. プロフィール写真

- ビジネスカジュアル（スーツまたはジャケット着用）の顔写真を使用する
- 背景は無地または落ち着いた色合い
- 笑顔で正面を向いた写真が好印象
- スマートフォンの自撮りではなく、誰かに撮ってもらうか写真館で撮影する
- 解像度400x400ピクセル以上を推奨

### 1-2. バナー画像

- デフォルトのまま放置しない
- 「転職先の業界に関連するイメージ」「都市の風景」「シンプルな抽象画像」等を設定する
- Canva等の無料ツールで作成可能（推奨サイズ: 1584x396ピクセル）

### 1-3. Headline（見出し）

プロフィール名の直下に表示される最重要テキスト。検索結果にも表示されるため、キーワードを意識して書く。

**書き方のルール**:
- 現在の肩書きだけでなく「何ができるか」「何を求めているか」を含める
- キーワード（Project Management, Stakeholder Management, Public Sector等）を入れる
- 120文字以内（全角の場合は半分程度）

### 1-4. About（自己紹介文）

プロフィールの「概要」欄。最大2,600文字記入可能だが、最初の3行（約300文字）がプレビュー表示されるため、冒頭に最も重要な情報を置く。

**書き方のルール**:
- 1段落目: 「私は何者か」と「何ができるか」を3行以内で
- 2段落目: 具体的な実績（数字を含める）
- 3段落目: 転職で求めていること
- 4段落目: スキルキーワード（箇条書き）
- 読みやすさのために改行を多用する

### 1-5. Experience（職務経歴）

公務員経歴を「民間の言葉」で記述する。公務員用語をそのまま書いても、民間の採用担当者やリクルーターには伝わらない。

**書き方のルール**:
- 各ポジションに3〜5個のバレットポイントを付ける
- 動詞で始める（Led, Managed, Coordinated, Delivered, Analyzed等）
- 数字を入れる（予算額、人数、件数、期間等）
- 「公務員用語 → 民間翻訳」を施す

### 1-6. Skills（スキル）

リクルーターが候補者を検索する際に使用するキーワード。最大50個設定可能。

### 1-7. Education（学歴）

大学名・学部・卒業年を記載。在学中の活動や取得資格も追記可能。

### 1-8. Certifications（資格）

TOEIC、英検、その他の資格を記載。スコアや取得年も入れる。

### 1-9. Recommendations（推薦文）

元上司や同僚からの推薦文があると信頼性が大幅に向上する。退職前に依頼しておくのが理想。

## 7. LinkedInでスカウトされやすいプロフィールの特徴

リクルーターが候補者を探す際のLinkedIn検索の仕組みを理解し、以下の7つの特徴を押さえる。

### 特徴1: 「Open to Work」を設定している

- LinkedInの設定画面で「求職中」のステータスをオンにする
- 「リクルーターにのみ表示」を選択すれば、現職にバレずに求職活動ができる
- 希望職種・希望勤務地・希望開始時期を具体的に設定する

### 特徴2: Headlineにキーワードが含まれている

- リクルーターは「Project Management」「Public Sector」「Government」等のキーワードで検索する
- Headlineにこれらのキーワードが含まれていないと検索結果に表示されない
- 「○○省 課長補佐」ではなく「Project Management | Public Sector | Government Relations」と書く

### 特徴3: Aboutセクションが充実している

- 空欄のプロフィールはスカウト対象から外れる
- 最低300文字以上記入する
- キーワードを自然に含めて記述する

### 特徴4: Experienceに具体的な成果が書かれている

- 「○○に従事」ではなく「○○を達成」と書く
- 数字を含める（予算額、件数、人数等）
- 動詞で始める（Led, Managed, Delivered, Achieved等）

### 特徴5: Skillsが30個以上設定されている

- 前述の30選を参考に設定する
- 同僚や知人にEndorsement（推薦）をもらうとランキングが上がる

### 特徴6: Connectionsが500人以上

- 500人以上になると「500+」と表示され、信頼性が高まる
- 同僚、大学の同期、セミナーで出会った人等を積極的につながる
- つながりリクエストには必ずメッセージを添える

### 特徴7: 定期的にアクティビティがある

- 週1回以上、記事のシェアやいいね、コメントを行う
- 自分の専門領域（行政、公共政策、テクノロジー等）に関する投稿をする
- アクティブなアカウントは検索結果の上位に表示されやすい

## 8. 直接応募の具体的手順

外資IT企業への応募は、エージェントを経由するよりも直接応募の方が書類通過率が高いケースが多い。以下の手順で応募する。

### 手順1: 求人を見つける

**LinkedInの求人検索を使う**:
1. LinkedInの上部メニューから「Jobs（求人）」をクリック
2. 検索窓にポジション名を入力（例: "Project Manager", "Business Development"）
3. フィルターで「Location: Japan」「Experience Level: Mid-Senior」等を設定
4. 「Easy Apply（簡単応募）」フィルターをオンにすると、LinkedIn上で直接応募できる求人のみ表示される

**企業の採用ページを直接確認する**:
1. 志望企業のコーポレートサイトにアクセス
2. 「Careers」「採用情報」「Join Us」等のページを探す
3. 日本オフィスのポジションを絞り込む
4. 応募フォームから直接応募する

### 手順2: 応募書類を準備する

- **レジュメ（英文）**: 前章のテンプレートを使って作成。PDF形式で保存
- **カバーレター**: 企業ごとにカスタマイズ。なぜこの企業か、どう貢献できるかを明記
- **LinkedInプロフィール**: 本ガイドに沿って最適化済みの状態にしておく

### 手順3: Easy Applyで応募する

1. 求人ページの「Easy Apply」ボタンをクリック
2. LinkedInに登録済みの情報（プロフィール）が自動で入力される
3. レジュメのPDFをアップロードする（自動入力に頼らず必ず添付する）
4. 追加の質問（ビザステータス、希望年収等）に回答する
5. 「Submit（送信）」をクリック

### 手順4: 企業の採用ページから応募する

1. 採用ページの応募フォームにアクセス
2. 基本情報（氏名、連絡先、学歴）を入力
3. レジュメをPDF形式でアップロード
4. カバーレターを添付（任意の場合も必ず添付する）
5. LinkedInプロフィールのURLを記入する欄があれば記入する
6. 応募ボタンをクリック

### 手順5: 応募後のフォローアップ

1. 応募した翌日〜3日後に、その企業の採用担当者やHiring ManagerをLinkedInで探す
2. つながりリクエストを送る（メッセージテンプレートは後述）
3. 1週間経っても返信がない場合、InMailまたはメッセージで丁寧にフォローする
4. しつこくならない範囲で（最大2回まで）

### 手順6: 応募管理

応募した企業・ポジション・日付・ステータスを一覧で管理する。以下の項目をスプレッドシート等で記録する:

| 項目 | 記入内容 |
|---|---|
| 企業名 | ○○ Japan |
| ポジション名 | Project Manager |
| 応募日 | 20XX/XX/XX |
| 応募経路 | LinkedIn Easy Apply / 企業HP / リファラル |
| ステータス | 応募済 / 書類通過 / 1次面接 / 最終面接 / 内定 / 不採用 |
| 次のアクション | フォローアップ送信 / 面接準備 |
| 備考 | 採用担当者名、面接日時等 |

### 推薦文（Recommendation）の依頼テンプレート

退職前に、元上司や信頼できる同僚にLinkedIn上の推薦文を依頼しておく。

```
○○さん

お世話になっております、○○です。

転職活動にあたり、LinkedInのプロフィールを充実させて
おりまして、もしよろしければ推薦文（Recommendation）を
書いていただけないでしょうか。

特に、以下のような点について一言いただけると
大変助かります:
- プロジェクト管理や調整業務における仕事ぶり
- チームへの貢献やリーダーシップ
- 仕事の品質やスピード

ご負担にならない範囲で構いません。
3〜5行程度の短いもので十分です。

お忙しいところ恐れ入りますが、ご検討いただければ
幸いです。
```

### 投稿コンテンツのアイデア

LinkedInで定期的に投稿すると、プロフィールの露出が増える。以下のようなテーマが反応を得やすい。

- キャリアチェンジの過程で学んだこと
- 公務員経験で培ったスキルが民間でどう活きるか
- 業界研究で発見した面白い知見
- 読んだ本やセミナーの感想（キャリア・ビジネス関連）
- 転職活動のTips

投稿頻度の目安は週1〜2回。長い投稿よりも、3〜5行の短い所感の方がエンゲージメントが高い傾向がある。
