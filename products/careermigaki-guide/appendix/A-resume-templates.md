# 付録A: 職務経歴書テンプレート

公務員経験を民間企業向けに「翻訳」した3パターンのテンプレートを掲載する。コピーして自分の経歴に書き換えて使ってほしい。

---

## パターン1: 公務員 → IT企業向け

### 職務要約

○○省にて約XX年間、政策立案・法令改正・予算編成等の行政実務に従事。総括補佐として省内横断プロジェクトを統括し、複数部署・外部ステークホルダーとの折衝を主導した。国会対応を通じて短納期での資料作成・合意形成を数百件単位で経験。これらのプロジェクトマネジメント力・ドキュメンテーション力を活かし、IT業界でのプロダクト推進・事業企画に貢献したい。

### 職務経歴

#### ○○省 ○○局 ○○課（20XX年X月〜20XX年X月）

**役職**: 総括補佐（課長補佐級）

**業務内容**:
- 省内横断の政策プロジェクト（関係部署5部署以上）の全体統括
  - 民間翻訳: 「5部門以上が関わるクロスファンクショナルプロジェクトのPMを担当」
- 年間予算約XX億円の予算案策定・財務省との折衝
  - 民間翻訳: 「年間XX億円規模の事業計画の策定と経営層への提案・承認獲得」
- 法令改正に伴う制度設計・パブリックコメント対応
  - 民間翻訳: 「新規サービス/プロダクトの要件定義とユーザーフィードバック反映」
- 国会質問への答弁資料作成（年間XX件以上、当日納期が大半）
  - 民間翻訳: 「経営層向けレポート・プレゼン資料を即日納期で年間XX件以上作成」

#### ○○省 ○○局 ○○課（20XX年X月〜20XX年X月）

**役職**: 係長

**業務内容**:
- ○○分野の調査・分析業務（データ収集・統計処理・報告書作成）
  - 民間翻訳: 「市場調査・データ分析に基づくレポーティングと施策提案」
- 関係機関（地方自治体・業界団体）との連絡調整
  - 民間翻訳: 「社外パートナー・クライアントとのリレーション構築と要件調整」
- 後輩指導・チームマネジメント（部下X名）
  - 民間翻訳: 「X名規模のチームリーダーとしてメンバー育成・業務配分を担当」

### スキル・資格

| カテゴリ | 内容 |
|---|---|
| 資格 | TOEIC 900 / 英検準1級 |
| PCスキル | Excel（VLOOKUP・ピボットテーブル・マクロ）、PowerPoint、Word |
| ツール | 基本的なBI/データ分析ツール（学習中の場合は「○○を自学中」と記載） |
| 語学 | 英語（ビジネスレベル：読み書き可、会話は実務での使用機会は限定的） |

### 自己PR

**プロジェクト推進力**: 総括補佐として、省内5部署以上が関わる横断プロジェクトを複数統括してきた。利害が対立する局面でも、各部署のニーズを丁寧にヒアリングし、落としどころを見つけて合意形成に導くことを得意としている。

**スピードと正確性の両立**: 国会対応では、当日中に正確な答弁資料を仕上げることが求められた。限られた時間の中で関係者から情報を集め、論点を整理し、読み手に伝わる資料に仕上げる力は、IT業界のスピード感にも十分適応できると考えている。

**ドキュメンテーション力**: 法令文書・政策資料・プレゼン資料など、読み手や目的に応じた文書作成を日常的に行ってきた。正確さと分かりやすさの両立を常に意識しており、この力は仕様書・企画書・社内ドキュメントの作成に直結する。

## パターン2: 公務員 → コンサルティングファーム向け

### 職務要約

○○省にて約XX年間、政策企画・制度設計・ステークホルダーマネジメントに従事。総括補佐として年間予算XX億円規模のプロジェクトを統括し、複雑な利害関係者間の合意形成を主導してきた。国会対応で培った論点整理力・資料作成力、行政特有の規制・制度に関する深い知見を活かし、パブリックセクター領域を中心としたコンサルティング業務に貢献したい。

### 職務経歴

#### ○○省 ○○局 ○○課（20XX年X月〜20XX年X月）

**役職**: 総括補佐（課長補佐級）

**プロジェクト実績**:

**[PJ1] ○○制度改革プロジェクト（期間: X年、関係者: XX名規模）**
- 課題: ○○分野における既存制度の運用上の問題点が顕在化
- 役割: プロジェクト全体統括（調査設計・関係者調整・報告書取りまとめ）
- アプローチ:
  - 現状分析: 定量データ（○○統計）と定性データ（関係者ヒアリングXX件）を統合分析
  - 制度設計: 3つの改革オプションを策定し、費用対効果と実現可能性の二軸で評価
  - 合意形成: 関係省庁XX省、業界団体XX団体との調整会議をXX回開催
- 成果: 制度改正案を取りまとめ、翌年度の法改正に反映

**[PJ2] ○○予算編成プロジェクト（年間予算約XX億円）**
- 課題: 財政制約下での予算確保と事業の優先順位付け
- 役割: 予算案の策定、財務省主計局との折衝
- アプローチ:
  - 全XX事業のROI分析とスコアリング
  - 財務省向けエビデンスブック（データ・事例・ロジック）の作成
  - 3回の査定ラウンドでの折衝と修正
- 成果: 要求額の約XX%を確保（省内平均を上回る獲得率）

#### ○○省 ○○局 ○○課（20XX年X月〜20XX年X月）

**役職**: 係長

**プロジェクト実績**:

**[PJ3] ○○調査・政策提言プロジェクト**
- 課題: ○○分野の実態把握と政策オプションの提示
- 役割: 調査設計・分析・報告書作成
- 成果: 調査報告書が政策決定の基礎資料として採用

### スキル・資格

| カテゴリ | 内容 |
|---|---|
| 資格 | TOEIC 900 / 英検準1級 |
| 分析スキル | 政策分析、費用対効果分析、ステークホルダー分析 |
| ドキュメント | 政策提言書、エビデンスブック、プレゼン資料 |
| 業界知見 | ○○分野の規制・制度・業界構造に精通 |
| 語学 | 英語（読解・文書作成はビジネスレベル） |

### 自己PR

**構造化された問題解決力**: 政策立案のプロセスでは、(1)現状分析 → (2)課題特定 → (3)オプション策定 → (4)評価・選定 → (5)実行計画、というフレームワークを日常的に使ってきた。これはコンサルティングにおける問題解決アプローチと本質的に同じであり、即戦力として活かせると考えている。

**パブリックセクターの深い知見**: 行政の意思決定プロセス・予算サイクル・関係者マップを実務レベルで熟知している。コンサルティングファームのパブリックセクター案件において、クライアントの内部力学を理解した上での提案が可能。

**高圧環境下での成果創出**: 国会対応では、予測不能なタイミングで高品質な資料が求められる。この環境で培った「短時間で論点を整理し、ロジカルなアウトプットを出す力」は、タイトなプロジェクトスケジュールの中でも発揮できる。

## パターン3: 公務員 → 外資IT向け（英文レジュメ付き）

### 職務要約（日本語版）

○○省にて約XX年間、政策立案・省内横断プロジェクトの統括に従事。総括補佐として、複数ステークホルダーとの折衝・合意形成を主導し、国会対応を通じてハイプレッシャー下での迅速な意思決定力を磨いてきた。TOEIC 900・英検準1級の英語力を持ち、外資IT企業の日本オフィスにおけるプロジェクトマネジメント・カスタマーサクセス・事業開発等のポジションで貢献したい。

### 自己PR（日本語版）

**グローバル環境への適応力**: TOEIC 900・英検準1級の英語力に加え、海外の行政機関との情報交換や国際会議への参加経験がある（該当する場合）。外資IT企業の日本オフィスでは日本語が主な業務言語であるため、英語力は「本社との連携」「グローバルプロジェクトへの参画」で差別化要因となる。

**行政クライアントへの橋渡し力**: 外資IT企業の日本法人にとって、官公庁・自治体は重要なクライアントセグメントである。行政の意思決定プロセスや調達制度を熟知していることは、パブリックセクター営業・カスタマーサクセスにおいて大きなアドバンテージとなる。

**変化への適応力と学習意欲**: 公務員として複数の部署を経験し、異動のたびに新しい分野をキャッチアップしてきた。技術領域への知的好奇心が強く、IT業界の学習コストにも積極的に向き合える。

## 公務員用語 → 民間翻訳 早見表

職務経歴書を書く際に、以下の翻訳表を参考にしてほしい。公務員の業務はそのまま書くと民間の採用担当者に伝わりにくい。「要は何をしたのか」を民間の言葉で表現することが重要。

| 公務員用語 | 民間向け翻訳 |
|---|---|
| 総括補佐 | プロジェクトマネージャー / シニアマネージャー |
| 法令改正の企画・立案 | 新規プロダクト/サービスの要件定義・設計 |
| 予算要求・財務省折衝 | 事業計画策定・経営層への予算獲得提案 |
| 国会答弁資料作成 | エグゼクティブ向けブリーフィング資料作成 |
| パブリックコメント対応 | ユーザーフィードバック収集・プロダクト改善 |
| 関係省庁との調整 | クロスファンクショナルなステークホルダー調整 |
| 審議会の運営 | 外部アドバイザリーボードの企画・運営 |
| 通達・通知の発出 | ガイドライン・社内ポリシーの策定・展開 |
| 地方自治体との連携 | 地域パートナー/クライアントとの協業 |
| 業界団体へのヒアリング | マーケットリサーチ・顧客インタビュー |
| 白書・報告書の執筆 | リサーチレポート・ホワイトペーパーの執筆 |
| 係長（チームリーダー） | チームリード / サブマネージャー |
| 課長 | ディレクター / 部門長 |
| 局長 | VP / 事業本部長 |
| 出向（他省庁・自治体） | ローテーション / クロスファンクショナル異動 |
| 定員管理・人事配置 | リソースプランニング・組織設計 |
