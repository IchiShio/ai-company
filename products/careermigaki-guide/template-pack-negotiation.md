# 年収交渉テンプレート集

公務員からの転職で最も戸惑うのが年収交渉。公務員には「交渉」という概念がほぼ存在しないため、多くの人がオファーをそのまま受け入れてしまう。しかし、特に外資系企業では交渉は「するのが当然」という文化がある。適切に交渉すれば、基本給50〜100万円のアップ、サインオンボーナスの追加、SOの増額といった成果を得られる可能性がある。

この章では、年収交渉に必要なすべてのテンプレートを提供する。

## 2. 年収交渉メールテンプレート5パターン

### テンプレート1: 基本給の増額を依頼（日本語）

```
件名: オファー条件についてのご相談

○○株式会社
人事部 ○○様

お世話になっております。○○です。

この度は○○ポジションのオファーをいただき、
誠にありがとうございます。御社で働くことへの
期待がますます高まっております。

オファー内容を拝見し、大変魅力的な条件だと
感じておりますが、基本給について一点ご相談を
させていただけないでしょうか。

ご提示いただいた基本給は○○万円ですが、以下の
理由から○○万円〜○○万円でのご検討をお願い
できればと存じます。

1. 現職の年収が○○万円であり、転職に伴う
   リスクを考慮すると、現在の水準からの
   上昇幅が限定的であること

2. 同業界・同職種の市場相場（OpenWork、
   Glassdoor等の公開データ）と照らし合わせ
   ると、○○万円〜○○万円がミッドレンジで
   あること

3. 面接でもお話しした通り、XX年間の
   プロジェクトマネジメント経験と
   パブリックセクター知見は、このポジション
   において即戦力として機能すると考えている
   こと

もちろん、御社の給与テーブルやご事情もあるかと
存じますので、柔軟にご相談させていただければ
幸いです。

御社で長期的に成果を出していく覚悟は変わり
ませんので、ぜひ前向きにご検討いただけますと
ありがたいです。

何卒よろしくお願い申し上げます。

○○
```

## 3. 口頭での交渉スクリプト3パターン

### スクリプト1: 基本給の交渉（対面/電話）

```
【切り出し】
「この度はオファーをいただき、本当にありがとうございます。
御社で働けることにとても期待しています。」

【本題への移行】
「一点、基本給についてご相談させていただけないでしょうか。
交渉をさせていただくことに恐縮ですが、率直にお話しさせてください。」

【根拠の提示】
「ご提示いただいた○○万円は魅力的な条件ですが、
市場データを調べたところ、同じレベルのポジションの
ミッドレンジは○○万円〜○○万円でした。
また、面接でもお話しした通り、私のXX年間の
プロジェクトマネジメント経験とパブリックセクター知見は、
このポジションで即戦力になると考えています。」

【具体的な希望を伝える】
「これらを踏まえ、○○万円でのご検討をお願いできれば
ありがたいのですが、いかがでしょうか。」

【柔軟さを示す】
「もちろん、御社の給与テーブルのご事情もあるかと思いますので、
柔軟にお話しさせていただければと思います。
基本給が難しければ、サインオンボーナスや株式報酬での
調整も選択肢として考えています。」

【締め】
「いずれにしても、御社への入社意欲は変わりません。
ぜひ前向きにご検討いただければ幸いです。」
```

### スクリプト2: 複数オファーがある場合の交渉

```
【切り出し】
「オファーをいただきありがとうございます。
御社は第一志望であり、ぜひ入社したいと考えています。」

【状況の開示】
「ただ、正直に申し上げると、他社からもオファーを
いただいている状況です。条件面では他社の方が
○○の点で高い提示をいただいています。」

【誠意を示す】
「しかし、私は事業内容やチームの環境を含めて
総合的に判断したいと考えており、条件だけで
決めるつもりはありません。御社が第一志望である
ことに変わりはありません。」

【具体的な依頼】
「その上で、基本給を○○万円にご調整いただく
ことは可能でしょうか。そうしていただければ、
他社のオファーを辞退して御社に入社する決断が
できます。」

【注意点】
「具体的な他社名は申し上げられませんが、
同業界の○○系企業です。
嘘の話をしてもすぐにバレますので、
事実をそのままお伝えしています。」
```

### スクリプト3: 交渉が難しいと言われた場合の切り返し

```
【相手の発言を受け止める】
「ご事情は承知いたしました。給与テーブルの中で
ご対応いただけるところまでご検討いただいたこと、
感謝しております。」

【代替案を提示する】
「基本給の調整が難しい場合、以下のような代替案は
ご検討いただけますでしょうか。」

「1. サインオンボーナスの追加
     基本給は据え置きで、入社時の一時金として
     ○○万円をいただけないでしょうか。」

「2. 初年度の評価時期の前倒し
     通常12ヶ月後の昇給レビューを、6ヶ月後に
     前倒しで実施いただくことは可能でしょうか。
     成果を早期に出す自信がありますので、
     結果で判断していただければと思います。」

「3. 株式報酬の増額
     キャッシュでの調整が難しい場合、
     SO/RSUの追加付与は検討可能でしょうか。
     長期的なコミットメントを示す形にもなると
     考えています。」

【最終的な姿勢】
「いずれかの形でご対応いただければ、
喜んでオファーをお受けいたします。
ご検討よろしくお願いいたします。」
```

## 5. 交渉で使ってはいけないNG表現リスト

以下の表現は、交渉の場で使うと印象を大きく損なう。絶対に避けること。

### NG表現1: 脅し系

| NG表現 | 理由 | 代替表現 |
|---|---|---|
| 「この条件では辞退します」 | 最後通牒は関係を壊す | 「ご検討いただけると嬉しいです」 |
| 「他社はもっと出してくれます」 | 比較で圧力をかけるのはNG | 「他社からもオファーをいただいており、総合的に判断したい」 |
| 「この条件なら他に行きます」 | 交渉ではなく脅迫になる | 「御社が第一志望ですが、条件面でご相談できれば」 |

### NG表現2: 感情系

| NG表現 | 理由 | 代替表現 |
|---|---|---|
| 「生活が苦しいので」 | 個人的な事情は交渉材料にならない | 「市場相場と私のスキルを考慮すると」 |
| 「家のローンがあるので」 | 同上 | 「長期的に御社に貢献する前提で」 |
| 「これだけの経験があるのに」 | 恩着せがましい | 「この経験が御社でどう活きるかを踏まえ」 |
| 「公務員を辞めてまで来るのだから」 | 自己犠牲アピールは逆効果 | 「御社での成長機会に期待して」 |

### NG表現3: 無知系

| NG表現 | 理由 | 代替表現 |
|---|---|---|
| 「相場が分からないのですが」 | 準備不足と見られる | 事前に相場を調べた上で具体的な数字を提示 |
| 「とりあえず上げてほしい」 | 根拠がない | 具体的な金額と理由を提示 |
| 「いくらでもいいです」 | 自分の価値を理解していない | 「○○万円〜○○万円を希望します」 |

### NG表現4: タイミング系

| NG表現 | 理由 | 代替表現 |
|---|---|---|
| （1次面接で）「年収はいくらですか」 | 早すぎる。選考の後半以降で聞くべき | 面接官から聞かれるまで待つ |
| （内定前に）「条件を上げてくれたら受けます」 | 内定後に交渉するのが原則 | 内定後に改めてご相談させてください |

### NG表現5: 嘘・誇張系

| NG表現 | 理由 | 代替表現 |
|---|---|---|
| 「他社から○○万円のオファーがあります」（嘘） | バレた場合、内定取り消しのリスク | 事実のみを伝える |
| 「現年収は○○万円です」（盛り） | 前職調査で発覚する可能性 | 源泉徴収票ベースの正確な金額を伝える |

## 7. 年収交渉の心構え（公務員出身者のためのマインドセット）

### 交渉は「わがまま」ではない

公務員の世界では、給与は俸給表で決まっており、交渉の余地がない。そのため、多くの公務員出身者は「年収交渉＝わがまま」「お金の話は品がない」と感じてしまう。

しかし、民間企業（特に外資）では、交渉は「自分の市場価値を正確に伝えるコミュニケーション」であり、ビジネスパーソンとしての基本スキルの一つと見なされている。交渉しないことは、むしろ「自分の価値を理解していない」とマイナスに解釈されることすらある。

### 交渉の3原則

**原則1: 根拠で語る**
「もっとほしい」ではなく「市場データと私のスキルに基づくと、この金額が妥当と考えます」と語る。感情ではなくデータが交渉の土台になる。

**原則2: win-winを目指す**
交渉は相手を打ち負かすものではない。「私が長期的に成果を出すために、双方が納得できる条件を見つけたい」という姿勢で臨む。

**原則3: 最悪のケースを受け入れる準備をする**
交渉が不調に終わった場合に「それでも入社するか」を事前に決めておく。撤退ラインが明確でないと、交渉に軸がなくなる。

### 交渉できるタイミング

- 内定（オファーレター受領）後が交渉のタイミング
- 選考中に年収の話を持ち出すのは時期尚早
- 回答期限前に交渉を完了させること
- 一度受諾した後の再交渉は原則不可

### 交渉で最も大切なこと

交渉の最中も最後も、「この人と一緒に働きたい」と思ってもらうことを忘れないこと。条件面で100%の成功を収めたとしても、交渉の過程で相手に嫌な思いをさせてしまっては、入社後の関係に影響する。礼儀正しさと感謝の姿勢を最後まで貫くこと。
